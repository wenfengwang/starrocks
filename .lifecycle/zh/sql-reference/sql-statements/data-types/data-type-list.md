# 数据类型概述

StarRocks 支持以下数据类型：[数值型](#numeric)、[字符串型](#string)、[日期型](#date)、[半结构化数据](#semi-structured)和[其他类型](#others)。您可以在创建表时定义这些类型的列，将数据加载到这些列，以及查询这些数据。

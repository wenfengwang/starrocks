# 展示当前运行中的查询

## 说明

展示当前在查询队列中正在执行或等待的所有查询信息。该功能从 v3.1.4 版本开始支持。

:::小贴士

此操作不需要特定权限。

:::

## 语法

```SQL
SHOW RUNNING QUERIES
```

## 返回值

- QueryId：查询的唯一标识符。
- ResourceGroupId：查询所触及的资源组的标识符。若没有触及到用户定义的资源组，则显示为“-”。
- StartTime：查询开始执行的时间。
- PendingTimeout：处于等待状态的查询在队列中超时的时间点。
- QueryTimeout：查询执行超时的时间点。
- State：查询的队列状态，"PENDING" 表示查询正在等待中，"RUNNING" 表示查询正在执行中。
- Slots：查询请求的逻辑资源量，目前默认为1。
- Frontend：初始化查询的前端（FE）节点。
- FeStartTime：初始化查询的前端（FE）节点的启动时间。

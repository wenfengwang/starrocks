# 添加秒数

## 描述

### 语法

```Haskell
DATETIME SECONDS_ADD(DATETIME expr1,INT expr2)
```

向日期中添加特定的时间间隔，单位为秒。

expr1 参数应为有效的日期时间表达式。

expr2 参数为您想要增加的秒数。

## 示例

```Plain
select seconds_add('2010-11-30 23:50:50', 2);
+---------------------------------------+
| seconds_add('2010-11-30 23:50:50', 2) |
+---------------------------------------+
| 2010-11-30 23:50:52                   |
+---------------------------------------+

select seconds_add('2010-11-30', 2);
+------------------------------+
| seconds_add('2010-11-30', 2) |
+------------------------------+
| 2010-11-30 00:00:02          |
+------------------------------+
```

## 关键字

SECONDS_ADD，ADD

# 小时

## 描述

返回指定日期的小时数。返回值介于0到23之间。

日期参数必须为 DATE 或 DATETIME 类型。

## 语法

```Haskell
INT HOUR(DATETIME|DATE date)
```

## 示例

```Plain
MySQL > select hour('2018-12-31 23:59:59');
+-----------------------------+
| hour('2018-12-31 23:59:59') |
+-----------------------------+
|                          23 |
+-----------------------------+
```

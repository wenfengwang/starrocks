# 年份

## 描述

提取日期中的年份部分，返回的值介于 1000 至 9999 之间。

日期参数必须为 DATE 或 DATETIME 类型。

## 语法

```Haskell
INT YEAR(DATETIME date)
```

## 示例

```Plain
MySQL > select year('1987-01-01');
+-----------------------------+
| year('1987-01-01 00:00:00') |
+-----------------------------+
|                        1987 |
+-----------------------------+
```

YEAR

# ISO周

## 描述

返回指定日期所在年份的ISO标准周数，该数值为一个介于1到53之间的整数。

## 语法

```Haskell
INT WEEK_ISO(DATETIME date)
```

## 参数

日期：您想要转换的日期。该日期必须是DATE或DATETIME类型。

## 示例

以下示例返回2008年2月20日00:00:00所在的ISO标准周数：

```SQL
MySQL > select week_iso ('2008-02-20 00:00:00');
+-----------------------------------+
| week_iso('2008-02-20 00:00:00')   |
+-----------------------------------+
|                                 8 |
+-----------------------------------+
```

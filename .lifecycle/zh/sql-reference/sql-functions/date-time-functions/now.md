# 当前时间，当前时间戳，本地时间，本地时间戳

## 描述

返回当前的日期和时间。自v3.1版本起，结果的精确度达到微秒级。

该函数可能会因不同的时区而返回不同的结果。更多信息，请参考[配置时区](../../../administration/timezone.md)。

## 语法

```Haskell
DATETIME NOW()
```

## 示例

```Plain
MySQL > select now();
+---------------------+
| now()               |
+---------------------+
| 2019-05-27 15:58:25 |
+---------------------+

-- The result is accurate to the microsecond since v3.1.
MySQL > select now();
+----------------------------+
| now()                      |
+----------------------------+
| 2023-11-18 12:54:34.878000 |
+----------------------------+
```

## 关键字

NOW，now

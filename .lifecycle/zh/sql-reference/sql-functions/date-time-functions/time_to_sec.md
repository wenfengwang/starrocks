# 时间转换为秒

## 描述

将时间值转换成秒数。转换使用的公式如下：

小时 x 3600 + 分钟 x 60 + 秒数

## 语法

```Haskell
BIGINT time_to_sec(TIME time)
```

## 参数

time：该参数必须是 TIME 类型。

## 返回值

返回一个 BIGINT 类型的值。如果输入无效，将返回 NULL。

## 示例

```plain
select time_to_sec('12:13:14');
+-----------------------------+
| time_to_sec('12:13:14')     |
+-----------------------------+
|                        43994|
+-----------------------------+
```

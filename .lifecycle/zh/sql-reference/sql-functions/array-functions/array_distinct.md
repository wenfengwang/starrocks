# 数组去重

## 描述

从数组中移除重复的元素。

## 语法

```Haskell
array_distinct(array)
```

## 参数

array：需要移除重复元素的数组。只支持 ARRAY 数据类型。

## 返回值

返回一个数组。

## 使用说明

- 返回的数组中的元素可能与指定数组中的元素的排序顺序不同。

- 返回的数组中的元素与指定数组中的元素具有相同的数据类型。

## 示例

本节使用以下表格作为示例：

```plaintext
mysql> select * from test;

+------+---------------+

| c1   | c2            |

+------+---------------+

|    1 | [1,1,2]       |

|    2 | [1,null,null] |

|    3 | NULL          |

|    4 | [null]        |

+------+---------------+
```

从列 c2 中移除重复值。

```plaintext
mysql> select c1, array_distinct(c2) from test;

+------+----------------------+

| c1   | array_distinct(`c2`) |

+------+----------------------+

|    1 | [2,1]                |

|    2 | [null,1]             |

|    3 | NULL                 |

|    4 | [null]               |

+------+----------------------+
```

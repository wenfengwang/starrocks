# 返回 x 的正切

## 描述

计算并返回角度 x（以弧度为单位）的正切值。

## 语法

```Haskell
TAN(x);
```

## 参数

x：支持双精度浮点数（DOUBLE）数据类型。

## 返回值

返回一个双精度浮点数（DOUBLE）数据类型的值。

## 示例

```Plain
mysql> select tan(3.14);
+-----------------------+
| tan(3.14)             |
+-----------------------+
| -0.001592654936407223 |
+-----------------------+
1 row in set (0.12 sec)
```

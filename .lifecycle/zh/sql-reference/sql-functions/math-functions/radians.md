# 弧度

## 描述

将角度 x 转换为弧度。

## 语法

```Haskell
REDIANS(x);
```

## 参数

x：支持双精度浮点数（DOUBLE）数据类型。

## 返回值

返回双精度浮点数（DOUBLE）数据类型的值。

## 示例

```Plain
mysql> select radians(90);
+--------------------+
| radians(90)        |
+--------------------+
| 1.5707963267948966 |
+--------------------+
1 row in set (0.00 sec)
```

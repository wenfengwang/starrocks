# 圆周率

## 描述

返回圆周率（Pi）的值。

## 语法

```Plain
pi()
```

## 参数

无

## 返回值

以双精度浮点数（DOUBLE 类型）的形式返回圆周率（Pi）的值。

## 示例

```Plain
mysql> select pi();
+-------------------+
| pi()              |
+-------------------+
| 3.141592653589793 |
+-------------------+
1 row in set (0.00 sec)
```

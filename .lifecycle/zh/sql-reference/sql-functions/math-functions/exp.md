# 指数, 自然指数

## 描述

返回常数 e 的 x 次幂的值。这个函数被称为自然对数的底数函数。

## 语法

```SQL
EXP(x);
```

## 参数

x：指数值。支持 DOUBLE 类型。

## 返回值

返回 DOUBLE 数据类型的值。

## 示例

计算 e 的 3.14 次幂：

```Plaintext
mysql> select exp(3.14);
+--------------------+
| exp(3.14)          |
+--------------------+
| 23.103866858722185 |
+--------------------+
1 row in set (0.01 sec)
```

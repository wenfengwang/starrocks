# 双曲正切

## 描述

计算参数的双曲正切值。

此函数从v3.0版本开始支持。

## 语法

```Haskell
DOUBLE tanh(DOUBLE arg)
```

### 参数

arg：只能指定数值类型的参数。在计算双曲正切之前，此函数会先将数值转换为DOUBLE类型。

## 返回值

返回的是DOUBLE数据类型的值。

## 使用须知

如果输入的是非数值类型的参数，该函数将返回NULL。

## 示例

```Plain
mysql> select tanh(-1);
+---------------------+
| tanh(-1)            |
+---------------------+
| -0.7615941559557649 |
+---------------------+

mysql> select tanh(0);
+---------+
| tanh(0) |
+---------+
|       0 |
+---------+

mysql> select tanh(1);
+--------------------+
| tanh(1)            |
+--------------------+
| 0.7615941559557649 |
+--------------------+

mysql> select tanh("");
+----------+
| tanh('') |
+----------+
|     NULL |
+----------+
```

# 模数函数

## 描述

模数函数用于返回被除数除以除数后的余数。

## 语法

```SQL
mod(dividend, divisor)
```

## 参数

- 被除数：需要被除的数字。
- 除数：用于除的数字。

被除数和除数都支持以下数据类型：

- TINYINT
- SMALLINT
- INT
- BIGINT
- LARGEINT
- FLOAT
- DOUBLE
- DECIMALV2
- DECIMAL32
- DECIMAL64
- DECIMAL128

> **注意**
> `被除数`和`除数`必须数据类型一致。StarRocks执行隐式转换，如果它们的数据类型不一致。

## 返回值

返回一个与被除数相同数据类型的值。如果除数被指定为0，StarRocks 会返回 NULL。

## 示例

```Plain
mysql> select mod(3.14,3.14);
+-----------------+
| mod(3.14, 3.14) |
+-----------------+
|               0 |
+-----------------+

mysql> select mod(3.14, 3);
+--------------+
| mod(3.14, 3) |
+--------------+
|         0.14 |
+--------------+

select mod(11,-5);
+------------+
| mod(11, -5)|
+------------+
|          1 |
+------------+

select mod(-11,5);
+-------------+
| mod(-11, 5) |
+-------------+
|          -1 |
+-------------+
```

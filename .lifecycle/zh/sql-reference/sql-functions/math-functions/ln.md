# 自然对数函数 ln、dlog1 说明

## 描述

计算一个数的自然对数（以 e 为底数）。

## 语法

```SQL
ln(arg)
log(arg)
dlog1(arg)
```

## 参数

- arg：需要计算对数的数值。只支持 DOUBLE 数据类型。

> **注意**
> StarRocks 返回 NULL 如果 `arg` 被指定为负数或 0。

## 返回值

返回的值为 DOUBLE 数据类型。

## 示例

示例 1：计算数字 3 的自然对数。

```SQL
mysql> select ln(3);
+--------------------+
| ln(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

mysql> select dlog1(3);
+--------------------+
| dlog1(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

mysql> select log(3);
+--------------------+
| log(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

```

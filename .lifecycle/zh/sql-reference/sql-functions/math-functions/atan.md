# 反正切

## 描述

计算参数的反正切值。

## 语法

```Haskell
DOUBLE atan(DOUBLE arg)
```

### 参数

arg：只能指定一个数值。在计算反正切之前，此函数会先将数值转换成 DOUBLE 类型。

## 返回值

返回的是 DOUBLE 数据类型的值。

## 使用须知

如果输入的是非数值，该函数将返回 NULL。

## 示例

```Plain
mysql> select atan(1);
+--------------------+
| atan(1)            |
+--------------------+
| 0.7853981633974483 |
+--------------------+

mysql> select atan(0);
+---------+
| atan(0) |
+---------+
|       0 |
+---------+

mysql> select atan(-1);
+---------------------+
| atan(-1)            |
+---------------------+
| -0.7853981633974483 |
+---------------------+

mysql> select atan("");
+----------+
| atan('') |
+----------+
|     NULL |
+----------+
```

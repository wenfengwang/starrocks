# 度数

## 描述

将以弧度表示的角度 x 转换为度数。

## 语法

```SQL
DEGREES(x);
```

## 参数

x：角度，以弧度为单位。支持 DOUBLE 类型。

## 返回值

返回一个 DOUBLE 类型的数值。

## 示例

```Plaintext
mysql> select degrees(3.1415926535898);
+--------------------------+
| degrees(3.1415926535898) |
+--------------------------+
|        180.0000000000004 |
+--------------------------+
1 row in set (0.07 sec)
```

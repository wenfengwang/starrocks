# 平方

## 描述

计算一个数值的平方。

## 语法

```Haskell
square(arg)
```

### 参数

arg：你想要计算平方的数值。你只能指定一个数值类型。该函数在计算数值的平方之前，会先将其转换成DOUBLE类型的值。

## 返回值

返回的是DOUBLE数据类型的值。

## 使用说明

如果你指定的是非数值类型，则该函数会返回NULL。

## 示例

示例一：计算一个数值的平方。

```Plain
mysql>  select square(11);
+------------+
| square(11) |
+------------+
|        121 |
+------------+
```

示例二：计算一个非数值的平方。返回值为NULL。

```Plain
mysql>  select square('2021-01-01');
+----------------------+
| square('2021-01-01') |
+----------------------+
|                 NULL |
+----------------------+
```

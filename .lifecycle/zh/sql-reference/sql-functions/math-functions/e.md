# e

## 描述

返回常数 e（自然对数的底数）。

## 语法

```SQL
E();
```

## 参数

此函数不需要任何参数。

## 返回值

以双精度浮点数（DOUBLE）形式返回常数 e。

## 示例

```Plaintext
mysql> select e();
+-------------------+
| e()               |
+-------------------+
| 2.718281828459045 |
+-------------------+
1 row in set (0.01 sec)
```

# atan2 函数

## 描述

返回 x 除以 y 的反正切值，即 x/y 的反正切。两个参数的正负号用来确定结果所在的象限。

返回值的范围是 [-π, π]。

## 语法

```Haskell
ATAN2(x,y);
```

## 参数

x：支持的数据类型是 DOUBLE。

y：支持的数据类型是 DOUBLE。

## 返回值

返回 DOUBLE 类型的值。如果 x 或 y 是 NULL，则返回 NULL。

## 示例

```Plain
mysql> select atan2(-0.8,2);
+---------------------+
| atan2(-0.8, 2)      |
+---------------------+
| -0.3805063771123649 |
+---------------------+
1 row in set (0.01 sec)
```

# 功能列表

StarRocks 提供了一系列丰富的函数，以便于数据查询和分析。除了常用的函数，StarRocks 还支持半结构化的函数，如 **ARRAY**、**JSON**、**MAP** 和 **STRUCT** 函数。同时，它还支持高阶 [Lambda 函数](Lambda_expression.md)。如果这些函数无法满足您的业务需求，您可以使用 [Java UDF](JAVA_UDF.md) 来自定义函数。

您可以根据以下分类查找相应的函数：

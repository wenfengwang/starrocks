本教程将对以下三个客户端进行测试，您只需选择其中一个即可：

- mysql CLI：您可以在Docker环境或您的本机上运行它。
- DBeaver提供[社区版](https://dbeaver.io/download/)和专业版。
- [MySQL Workbench](https://dev.mysql.com/downloads/workbench/)

#### 配置客户端

<Tabs groupId="client">
<TabItem value="mysqlCLI" label="mysql CLI" default>


使用mysql CLI最简单的方法是在StarRocks容器starrocks-fe中运行它：

```bash
docker exec -it quickstart \
mysql -P 9030 -h 127.0.0.1 -u root --prompt="StarRocks > "
```

:::提示 所有docker-compose命令都必须在包含docker-compose.yml文件的目录中运行。:::

如果您想要安装**mysql客户端**CLI，请展开下方的mysql客户端安装指南：

<details>


<summary>mysql client install</summary>


- macOS：如果您使用Homebrew且不需要MySQL服务器，运行brew install mysql即可安装CLI。
- Linux：检查您的包管理器是否提供mysql客户端。例如，执行yum install mariadb。
- Microsoft Windows：安装[MySQL Community Server](https://dev.mysql.com/downloads/mysql/)并运行提供的客户端，或者在WSL中运行`mysql`。

</details>

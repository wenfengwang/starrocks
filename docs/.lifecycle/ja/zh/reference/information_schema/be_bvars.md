# be_bvars

`be_bvars` は bRPC の統計情報を提供します。RPC の遅延、QPS、および一部の StarRocks コンポーネントの統計情報を確認できます。

`be_bvars` は以下のフィールドを提供します：

| **フィールド** | **説明**                          |
| -------------- | --------------------------------- |
| BE_ID          | bvar が存在する BE の ID。         |
| NAME           | bvar の名前。                     |
| DESC           | bvar の説明、統計情報の値を含む。 |

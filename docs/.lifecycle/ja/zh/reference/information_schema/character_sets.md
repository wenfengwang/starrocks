# character_sets

`character_sets` は使用可能な文字セットを識別するために使用されます。

`character_sets` には以下のフィールドがあります：

| **フィールド**         | **説明**                           |
| -------------------- | ---------------------------------- |
| CHARACTER_SET_NAME   | 文字セットの名前。                 |
| DEFAULT_COLLATE_NAME | 文字セットのデフォルトの照合順序。 |
| DESCRIPTION          | 文字セットの説明。                 |
| MAXLEN               | 1文字を格納するために必要な最大バイト数。 |

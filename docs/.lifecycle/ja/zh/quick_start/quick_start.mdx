# クイックスタートガイド

このクイックスタートガイドは、小規模な StarRocks 環境を迅速に使用するために役立ちます。あなたが立ち上げる予定のクラスターは、StarRocks の動作原理を学ぶのに適していますが、大規模なデータセットの分析やパフォーマンステストには適していません。

展開を拡大する場合は、[StarRocks のデプロイ](../deployment/deployment_overview.md)を参照してください。

# tan

## 機能

引数 `x` の正接を計算します。`x` はラジアン単位です。

## 構文

```Haskell
TAN(x);
```

## 引数の説明

`x`: サポートされるデータ型は DOUBLE です。

## 戻り値の説明

戻り値のデータ型は DOUBLE です。

## 例

```Plain Text
mysql> select tan(3.14);
+-----------------------+
| tan(3.14)             |
+-----------------------+
| -0.001592654936407223 |
+-----------------------+
1行がセットされました (0.12 秒)
```

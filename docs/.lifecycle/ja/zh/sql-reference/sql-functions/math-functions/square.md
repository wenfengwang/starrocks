# SQUARE

## 機能

引数の平方を計算します。

## 文法

```sql
SQUARE(arg)
```

### 引数の説明

`arg`：平方を計算する必要がある引数で、サポートされているデータ型は数値型です。実装時にはこの引数をDOUBLE型に変換して計算します。

## 戻り値の説明

戻り値のデータ型はDOUBLEです。

## 注意点

引数が数値型でない場合、戻り値はNULLになります。

## 例

例1：数値型の引数の平方を計算します。

```Plain Text
mysql>  select square(11);
+------------+
| square(11) |
+------------+
|        121 |
+------------+
```

例2：引数の型が数値型でない場合、戻り値はNULLになります。

```Plain Text
mysql>  select square('2021-01-01');
+----------------------+
| square('2021-01-01') |
+----------------------+
|                 NULL |
+----------------------+
```

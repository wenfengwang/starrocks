# ln、dlog1、log

## 機能

引数 `x` の自然対数を返します。底は e です。

## 文法

```Haskell
LN(x);
```

## 引数説明

`x`: 必須、サポートされるデータ型は DOUBLE。`x` は 0 より大きくなければならず、そうでない場合は NULL を返します。

## 戻り値の説明

戻り値のデータ型は DOUBLE です。

## 例

```Plain Text
mysql> select ln(2);
+--------------------+
| ln(2)              |
+--------------------+
| 0.6931471805599453 |
+--------------------+
1 行がセットされました (0.01 秒)

select dlog1(2);
+--------------------+
| dlog1(2)           |
+--------------------+
| 0.6931471805599453 |
+--------------------+
1 行がセットされました (0.00 秒)
```

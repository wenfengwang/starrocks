# tanh

## 機能

入力された数値の双曲線正接（ハイパボリックタンジェント）を計算します。

この関数はバージョン3.0からサポートされています。

## 構文

```Haskell
DOUBLE tanh(DOUBLE arg)
```

### 引数の説明

`arg`：数値を指定します。この関数は、入力された数値の双曲線正接を計算する前に、数値をDOUBLE型に変換します。

## 戻り値の説明

DOUBLE型の値を返します。

## 使用方法

引数が数値でない場合は、`NULL`を返します。

## 例

```Plain
mysql> select tanh(-1);
+---------------------+
| tanh(-1)            |
+---------------------+
| -0.7615941559557649 |
+---------------------+

mysql> select tanh(0);
+---------+
| tanh(0) |
+---------+
|       0 |
+---------+

mysql> select tanh(1);
+--------------------+
| tanh(1)            |
+--------------------+
| 0.7615941559557649 |
+--------------------+

mysql> select tanh("");
+----------+
| tanh('') |
+----------+
|     NULL |
+----------+
```

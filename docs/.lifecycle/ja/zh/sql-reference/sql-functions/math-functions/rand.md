# rand、random

## 機能

0（含む）から1（含まない）の間のランダムな浮動小数点数を返します。

## 文法

```Haskell
RAND(x);
```

## パラメータ説明

`x`: オプション。サポートされるデータ型は BIGINT です。`x` を指定した場合、再現可能なランダム数を返します。`x` を指定しない場合、完全にランダムな数を返します。

## 戻り値の説明

戻り値のデータ型は DOUBLE です。

## 例

```Plain Text
mysql> select rand();
+--------------------+
| rand()             |
+--------------------+
| 0.9393535880089522 |
+--------------------+
1 row in set (0.01 sec)

mysql> select rand(3);
+--------------------+
| rand(3)            |
+--------------------+
| 0.6659865964511347 |
+--------------------+
1 row in set (0.00 sec)
```

## キーワード

RAND, random

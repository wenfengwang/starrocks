# ASIN

## 機能

`x` のアークサイン（逆正弦）値を返します（単位はラジアン）。`x` はDOUBLE型の数値です。

## 文法

```Haskell
ASIN(x);
```

## パラメータ説明

`x`: 対応するデータ型はDOUBLEです。値の範囲は[-1,1]です。

## 戻り値の説明

戻り値のデータ型はDOUBLEです。入力値がNULLまたは`x`が[-1,1]の範囲を超える場合、NULLを返します。

## 例

```Plain Text
mysql> select asin(0.25);
+---------------------+
| asin(0.25)          |
+---------------------+
| 0.25268025514207865 |
+---------------------+
1行がセットされました (0.01秒)
```

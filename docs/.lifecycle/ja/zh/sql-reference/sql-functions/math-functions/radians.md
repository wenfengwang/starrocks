# ラジアン

## 機能

引数 `x` をラジアンに変換します。`x` は角度です。

## 文法

```Haskell
RADIANS(x);
```

## 引数の説明

`x`: サポートされるデータ型は DOUBLE です。

## 戻り値の説明

戻り値のデータ型は DOUBLE です。

## 例

```Plain Text
mysql> select radians(90);
+--------------------+
| radians(90)        |
+--------------------+
| 1.5707963267948966 |
+--------------------+
1行がセットされました (0.00 秒)
```

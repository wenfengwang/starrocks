# cos

## 機能

引数 `x` の余弦（コサイン）値を求めます。

## 文法

```Haskell
COS(x);
```

## 引数の説明

`x`: ラジアン単位の値で、サポートされるデータ型は DOUBLE です。

## 戻り値の説明

戻り値のデータ型は DOUBLE です。

## 例

```Plain Text
mysql> select cos(2);
+---------------------+
| cos(2)              |
+---------------------+
| -0.4161468365471424 |
+---------------------+
1 row in set (0.01 sec)
```

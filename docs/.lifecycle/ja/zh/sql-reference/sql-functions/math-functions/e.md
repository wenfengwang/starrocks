# e

## 機能

自然対数の底を返します。

## 構文

```Haskell
E();
```

## 引数説明

なし

## 戻り値の説明

戻り値のデータ型は DOUBLE です。

## 例

```Plain Text
mysql> select e();
+-------------------+
| e()               |
+-------------------+
| 2.718281828459045 |
+-------------------+
1行がセットされました (0.01 秒)
```

# degrees

## 機能

引数 `x` を度数に変換します。`x` はラジアンです。

## 文法

```Haskell
DEGREES(x);
```

## 引数の説明

`x`: 対応するデータ型は DOUBLE です。

## 戻り値の説明

戻り値のデータ型は DOUBLE です。

## 例

```Plain Text
mysql> select degrees(3.1415926535898);
+--------------------------+
| degrees(3.1415926535898) |
+--------------------------+
|        180.0000000000004 |
+--------------------------+
1行がセットされました (0.07 秒)
```

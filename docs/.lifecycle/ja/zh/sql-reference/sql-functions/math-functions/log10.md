# LOG10、DLOG10

## 機能

10を底とする`x`の対数を返します。

## 文法

```Haskell
LOG10(x);
```

## パラメータ説明

`x`: 対数を計算する数値で、必須です。サポートされるデータ型はDOUBLEです。`x`は0より大きくなければならず、そうでない場合はNULLを返します。

## 戻り値の説明

戻り値のデータ型はDOUBLEです。

## 例

```Plain Text
mysql> select log10(100);
+------------+
| log10(100) |
+------------+
|          2 |
+------------+
1行がセットされました (0.01秒)
```

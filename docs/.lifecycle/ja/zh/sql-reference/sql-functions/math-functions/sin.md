# sin

## 機能

引数 `x` の正弦を計算します。`x` はラジアン単位です。

## 文法

```Haskell
SIN(x);
```

## 引数説明

`x`: 対応するデータ型は DOUBLE です。

## 戻り値の説明

戻り値のデータ型は DOUBLE です。

## 例

```Plain Text
mysql> select sin(3.14);
+-----------------------+
| sin(3.14)             |
+-----------------------+
| 0.0015926529164868282 |
+-----------------------+
1 行がセットされました (0.21 秒)
```

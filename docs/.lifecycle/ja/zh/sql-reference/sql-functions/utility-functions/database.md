# データベース

## 機能

現在のセッションが使用しているデータベースを照会します。データベースが選択されていない場合は、nullを返します。

## 文法

```Haskell
database()
```

## パラメータ説明

この関数はパラメータを必要としません。

## 戻り値の説明

現在のセッションが使用しているデータベース名を返します。

## 例

```SQL
-- 目的のデータベースを選択します。
use db_test

-- 現在のデータベース名を返します。
select database();
+------------+
| DATABASE() |
+------------+
| db_test    |
+------------+
```

## 関連するSQL

[USE](../../sql-statements/data-definition/USE.md)：指定されたデータベースに切り替えます。

# coalesce

## 機能

左から右へと最初の非NULLの引数を返します。

## 文法

```Haskell
coalesce(expr1,...);
```

## 引数説明

`expr1`: 式はデータ型が互換性がある必要があります。そうでなければエラーが返されます。

## 戻り値の説明

戻り値のデータ型は `expr1` の型と一致します。

## 例

```Plain Text
mysql> select coalesce(3,NULL,1,1);
+-------------------------+
| coalesce(3, NULL, 1, 1) |
+-------------------------+
|                       3 |
+-------------------------+

mysql> select coalesce(NULL,2,1,1);
+-------------------------+
| coalesce(NULL, 2, 1, 1) |
+-------------------------+
|                       2 |
+-------------------------+
```

# データ型の概要

StarRocks は以下のデータ型をサポートしています：[数値型](#数値型)、[文字列型](#文字列型)、[日付型](#日付型)、[半構造化型](#半構造化型)、[その他の型](#その他の型)。テーブルを作成する際にこれらの型の列を指定し、その型のデータをテーブルにインポートしてクエリを実行することができます。

## 数値型

- [TINYINT](./TINYINT.md)
- [SMALLINT](./SMALLINT.md)
- [INT](./INT.md)
- [BIGINT](./BIGINT.md)
- [LARGEINT](./LARGEINT.md)
- [DECIMAL](./DECIMAL.md)
- [DOUBLE](./DOUBLE.md)
- [FLOAT](./FLOAT.md)
- [BOOLEAN](./BOOLEAN.md)
- PERCENTILE

## 文字列型

- [CHAR](./CHAR.md)
- [STRING](./STRING.md)
- [VARCHAR](./VARCHAR.md)
- [BINARY](./BINARY.md)

## 日付型

- [DATE](./DATE.md)
- [DATETIME](./DATETIME.md)

## 半構造化型

- [ARRAY](./Array.md)
- [JSON](./JSON.md)
- [MAP](./Map.md)
- [STRUCT](./STRUCT.md)

## その他の型

- [BITMAP](./BITMAP.md)
- [HLL](./HLL.md)

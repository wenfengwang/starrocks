これらの3つのクライアントは、このチュートリアルでテストされていますが、1つだけが必要です：

- mysql CLI: これはDocker環境またはあなたのマシンから実行できます。
- [DBeaver](https://dbeaver.io/download/)にはコミュニティバージョンとプロバージョンがあります。
- [MySQL Workbench](https://dev.mysql.com/downloads/workbench/)

#### クライアントの設定

<Tabs groupId="client">
<TabItem value="mysqlCLI" label="mysql CLI" default>

mysql CLIを最も簡単に使用する方法は、StarRocksコンテナ`starrocks-fe`から実行することです：

```bash
docker compose exec starrocks-fe \
mysql -P 9030 -h 127.0.0.1 -u root --prompt="StarRocks > "
```

:::tip
すべての`docker compose`コマンドは、`docker-compose.yml`ファイルが含まれているディレクトリから実行する必要があります。
:::

mysql CLIをインストールしたい場合は、以下の**mysql client install**を展開してください：

<details>

<summary>mysql client install</summary>

- macOS: Homebrewを使用しており、MySQLサーバーが不要な場合は`brew install mysql`を実行してCLIをインストールします。
- Linux: リポジトリシステムで`mysql`クライアントをチェックしてください。例えば、`yum install mariadb`です。
- Microsoft Windows: [MySQL Community Server](https://dev.mysql.com/downloads/mysql/)をインストールし、提供されるクライアントを実行するか、WSLから`mysql`を実行します。

</details>

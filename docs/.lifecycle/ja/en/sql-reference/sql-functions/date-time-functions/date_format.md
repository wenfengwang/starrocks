# date_format

## 説明

日付を指定されたフォーマットに従って文字列に変換します。現在、最大128バイトの文字列をサポートしています。返される値の長さが128バイトを超える場合、NULLが返されます。

## 構文

```Haskell
VARCHAR DATE_FORMAT(DATETIME date, VARCHAR format)
```

## パラメータ

- `date` パラメータは有効な日付または日付表現でなければなりません。

- `format` は日付または時刻の出力フォーマットを指定します。

使用可能なフォーマットは次のとおりです：

```Plain Text
%a | 短縮された曜日名 (Sun to Sat)
%b | 短縮された月名 (Jan to Dec)
%c | 数字で表された月名 (0-12)
%D | 月の日付を数字で表し、英語の接尾辞を伴う
%d | 月の日付を数字で表す (00-31)
%e | 月の日付を数字で表す (0-31)
%f | マイクロ秒
%H | 時間 (00-23)
%h | 時間 (01-12)
%I | 時間 (01-12)
%i | 分 (00-59)
%j | 年の日 (001-366)
%k | 時間 (0-23)
%l | 時間 (1-12)
%M | 月の完全な名前
%m | 数字で表された月名 (00-12)
%p | AMまたはPM
%r | 12時間制の時間 (hh:mm:ss AM or PM)
%S | 秒 (00-59)
%s | 秒 (00-59)
%T | 24時間制の時間 (hh:mm:ss)
%U | 週 (00-53) 日曜日が週の最初の日
%u | 週 (00-53) 月曜日が週の最初の日
%V | 週 (01-53) 日曜日が週の最初の日。%Xと共に使用。
%v | 週 (01-53) 月曜日が週の最初の日。%xと共に使用。
%W | 完全な曜日名
%w | 週の日 (日曜日=0 から 土曜日=6)
%X | 週の年、日曜日が週の最初の日。4桁の値。%Vと共に使用。
%x | 週の年、月曜日が週の最初の日。4桁の値。%vと共に使用。
%Y | 年。4桁の値。
%y | 年。2桁の値。
%% | %記号を表す。
```

## 例

```Plain Text
MySQL > select date_format('2009-10-04 22:23:00', '%W %M %Y');
+------------------------------------------------+
| date_format('2009-10-04 22:23:00', '%W %M %Y') |
+------------------------------------------------+
| Sunday October 2009                            |
+------------------------------------------------+

MySQL > select date_format('2007-10-04 22:23:00', '%H:%i:%s');
+------------------------------------------------+
| date_format('2007-10-04 22:23:00', '%H:%i:%s') |
+------------------------------------------------+
| 22:23:00                                       |
+------------------------------------------------+

MySQL > select date_format('1900-10-04 22:23:00', '%D %y %a %d %m %b %j');
+------------------------------------------------------------+
| date_format('1900-10-04 22:23:00', '%D %y %a %d %m %b %j') |
+------------------------------------------------------------+
| 4th 00 Thu 04 10 Oct 277                                   |
+------------------------------------------------------------+

MySQL > select date_format('1997-10-04 22:23:00', '%H %k %I %r %T %S %w');
+------------------------------------------------------------+
| date_format('1997-10-04 22:23:00', '%H %k %I %r %T %S %w') |
+------------------------------------------------------------+
| 22 22 10 10:23:00 PM 22:23:00 00 6                         |
+------------------------------------------------------------+

MySQL > select date_format('1999-01-01 00:00:00', '%X %V');
+---------------------------------------------+
| date_format('1999-01-01 00:00:00', '%X %V') |
+---------------------------------------------+
| 1998 52                                     |
+---------------------------------------------+

MySQL > select date_format('2006-06-01', '%d');
+------------------------------------------+
| date_format('2006-06-01 00:00:00', '%d') |
+------------------------------------------+
| 01                                       |
+------------------------------------------+

MySQL > select date_format('2006-06-01', '%%%d');
+--------------------------------------------+
| date_format('2006-06-01 00:00:00', '%%%d') |
+--------------------------------------------+
| %01                                        |
+--------------------------------------------+
```

## キーワード

DATE_FORMAT, DATE, FORMAT

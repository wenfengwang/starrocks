# utc_time

## 説明

現在のUTC時刻を'HH:MM:SS'形式の値として返します。

## 構文

```Haskell
TIME UTC_TIME()
```

## 例

```Plain Text
MySQL > select utc_time();
+----------------+
| utc_time()     |
+----------------+
| 12:31:18       |
+----------------+
```


## キーワード

UTC_TIME, UTC, TIME

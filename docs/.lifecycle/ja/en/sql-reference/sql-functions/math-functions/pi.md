# pi

## 説明

円周率の値を返します。

## 構文

```Plain
pi()
```

## パラメーター

N/A

## 戻り値

円周率の値をDOUBLE型の値として返します。

## 例

```Plain
mysql> select pi();
+-------------------+
| pi()              |
+-------------------+
| 3.141592653589793 |
+-------------------+
1 row in set (0.00 sec)
```

# conv

## 説明

数値 `x` をある数値基底システムから別の数値基底システムに変換し、結果を文字列値として返します。

## 構文

```Haskell
CONV(x,y,z);
```

## パラメータ

- `x`: 変換される数値。VARCHAR または BIGINT がサポートされています。
- `y`: 元の基底。TINYINT がサポートされています。
- `z`: 目標の基底。TINYINT がサポートされています。

## 戻り値

VARCHAR データ型の値を返します。

## 例

数値 8 を数値基底システム 2 から数値基底システム 10 に変換:

```Plain
mysql> select conv(8,10,2);
+----------------+
| conv(8, 10, 2) |
+----------------+
| 1000           |
+----------------+
1 行がセットされました (0.00 秒)
```

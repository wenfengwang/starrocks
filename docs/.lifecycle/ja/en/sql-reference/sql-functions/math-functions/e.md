# e

## 説明

定数 e（自然対数の底）を返します。

## 構文

```SQL
E();
```

## パラメーター

この関数はパラメータを取りません。

## 戻り値

DOUBLE 型として定数 e を返します。

## 例

```Plaintext
mysql> select e();
+-------------------+
| e()               |
+-------------------+
| 2.718281828459045 |
+-------------------+
1 row in set (0.01 sec)
```

# ln、dlog1、log

## 説明

数値の自然対数（底`e`）を計算します。

## 構文

```SQL
ln(arg)
log(arg)
dlog1(arg)
```

## パラメーター

- `arg`: 対数を計算したい値です。DOUBLE データ型のみサポートされています。

> **注記**
>
> StarRocksは、`arg`が負の数または0として指定された場合、NULLを返します。

## 戻り値

DOUBLE データ型の値を返します。

## 例

例1：3の自然対数を計算します。

```SQL
mysql> select ln(3);
+--------------------+
| ln(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

mysql> select dlog1(3);
+--------------------+
| dlog1(3)           |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

mysql> select log(3);
+--------------------+
| log(3)             |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

```

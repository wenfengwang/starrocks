# tan

## 説明

`x`のタンジェントを返します。ここで`x`はラジアン単位です。

## 構文

```Haskell
TAN(x);
```

## パラメーター

`x`: DOUBLE データ型をサポートします。

## 戻り値

DOUBLE データ型の値を返します。

## 例

```Plain
mysql> select tan(3.14);
+-----------------------+
| tan(3.14)             |
+-----------------------+
| -0.001592654936407223 |
+-----------------------+
1 row in set (0.12 sec)
```

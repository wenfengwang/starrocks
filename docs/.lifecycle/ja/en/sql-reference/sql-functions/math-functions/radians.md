# radians

## 説明

角度 `x` をラジアンに変換します。

## 構文

```Haskell
RADIANS(x);
```

## パラメーター

`x`: DOUBLE データ型をサポートします。

## 戻り値

DOUBLE データ型の値を返します。

## 例

```Plain
mysql> select radians(90);
+--------------------+
| radians(90)        |
+--------------------+
| 1.5707963267948966 |
+--------------------+
1 行がセットされました (0.00 秒)
```

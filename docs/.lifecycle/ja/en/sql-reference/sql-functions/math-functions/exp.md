# exp、dexp

## 説明

`e` の `x` 乗の値を返します。この関数は自然対数関数と呼ばれます。

## 構文

```SQL
EXP(x);
```

## パラメーター

`x`: べき乗の指数。DOUBLE 型がサポートされています。

## 戻り値

DOUBLE データ型の値を返します。

## 例

3.14 の `e` 乗の値を返します:

```Plaintext
mysql> select exp(3.14);
+--------------------+
| exp(3.14)          |
+--------------------+
| 23.103866858722185 |
+--------------------+
1行がセットになりました (0.01 秒)
```

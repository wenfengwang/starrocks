# sin

## 説明

`x`の正弦を返します。ここで`x`はラジアン単位です。

## 構文

```Haskell
SIN(x);
```

## パラメーター

`x`: DOUBLE データ型をサポートします。

## 戻り値

DOUBLE データ型の値を返します。

## 例

```Plain
mysql> select sin(3.14);
+-----------------------+
| sin(3.14)             |
+-----------------------+
| 0.0015926529164868282 |
+-----------------------+
1行がセットされました (0.21 秒)
```

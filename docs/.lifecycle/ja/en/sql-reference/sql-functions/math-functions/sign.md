# sign

## 説明

`x`の符号を返します。入力としての負の数、0、または正の数は、それぞれ`-1`、`0`、`1`として出力されます。

## 構文

```Haskell
SIGN(x);
```

## パラメーター

`x`: DOUBLE データ型をサポートします。

## 戻り値

FLOAT データ型の値を返します。

## 例

```Plain
mysql> select sign(3.14159);
+---------------+
| sign(3.14159) |
+---------------+
|             1 |
+---------------+
1 row in set (0.02 sec)
```

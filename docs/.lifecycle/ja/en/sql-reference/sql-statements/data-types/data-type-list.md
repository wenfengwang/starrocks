# データ型の概要

StarRocksは、次のデータ型をサポートしています：[数値](#numeric)、[文字列](#string)、[日付](#date)、[半構造化](#semi-structured)、および[その他](#others)。これらの型のカラムは、テーブルを作成する際に定義でき、これらのカラムにデータをロードし、データをクエリすることができます。

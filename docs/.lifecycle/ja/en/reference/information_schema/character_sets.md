# character_sets

`character_sets` は、利用可能な文字セットを特定します。

`character_sets` には以下のフィールドがあります：

| **フィールド**            | **説明**                                              |
| -------------------- | ------------------------------------------------------------ |
| CHARACTER_SET_NAME   | 文字セットの名前です。                                      |
| DEFAULT_COLLATE_NAME | 文字セットのデフォルト照合順序です。                 |
| DESCRIPTION          | 文字セットに関する説明です。                          |
| MAXLEN               | 1文字を格納するのに必要な最大バイト数です。 |

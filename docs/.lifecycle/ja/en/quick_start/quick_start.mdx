# クイックスタート

これらのクイックスタートガイドは、小規模なStarRocks環境での作業を開始するのに役立ちます。あなたが立ち上げるクラスターは、StarRocksの動作を学ぶには適していますが、大規模なデータセットの分析やパフォーマンステストには向いていません。

スケーラブルなデプロイメントについては、[StarRocksのデプロイメント](../deployment/deployment_overview.md)をご覧ください。

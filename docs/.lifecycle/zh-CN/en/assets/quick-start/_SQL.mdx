#### 在纽约市能否安全驾驶在能见度差的情况下？

让我们看看在能见度差（0到1.0英里之间）时发生车祸的次数。 要回答这个问题，请在 DATETIME 列上使用两个表的 JOIN。

```sql
SELECT COUNT(DISTINCT c.COLLISION_ID) AS 车祸数,
       truncate(avg(w.HourlyDryBulbTemperature), 1) AS 温度_F,
       truncate(avg(w.HourlyVisibility), 2) AS 能见度,
       max(w.HourlyPrecipitation) AS 降水量,
       date_format((date_trunc("hour", c.CRASH_DATE)), '%d %b %Y %H:%i') AS 时间
FROM crashdata c
LEFT JOIN weatherdata w
ON date_trunc("hour", c.CRASH_DATE)=date_trunc("hour", w.DATE)
WHERE w.HourlyVisibility BETWEEN 0.0 AND 1.0
GROUP BY 时间
ORDER BY 车祸数 DESC
LIMIT 100;
```

能见度差的情况下单小时发生的最高车祸数是129。 有多个因素需要考虑：

- 2014年2月3日是周一
- 8点是高峰时间
- 当时下雨（那个小时的降水量为0.12英寸）
- 温度为32华氏度（水的冰点）
- 能见度为0.25英里，纽约市正常的能见度为10英里

#### 在冰冷条件下驾驶怎么样？

水蒸气可以在华氏40度以下凝华成冰，这个查询查看了0到40华氏度之间的温度。

```sql
SELECT COUNT(DISTINCT c.COLLISION_ID) AS 车祸数,
       truncate(avg(w.HourlyDryBulbTemperature), 1) AS 温度_F,
       truncate(avg(w.HourlyVisibility), 2) AS 能见度,
       max(w.HourlyPrecipitation) AS 降水量,
       date_format((date_trunc("hour", c.CRASH_DATE)), '%d %b %Y %H:%i') AS 时间
FROM crashdata c
LEFT JOIN weatherdata w
ON date_trunc("hour", c.CRASH_DATE)=date_trunc("hour", w.DATE)
WHERE w.HourlyDryBulbTemperature BETWEEN 0.0 AND 40.5 
GROUP BY 时间
ORDER BY 车祸数 DESC
LIMIT 100;
```

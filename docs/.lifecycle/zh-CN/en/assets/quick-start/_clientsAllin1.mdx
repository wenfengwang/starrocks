这三种客户端都已经过本教程测试，您只需选择其中一个：

- mysql CLI：您可以从Docker环境或您的机器上运行此命令。

#### 配置客户端

<Tabs groupId="client">
<TabItem value="mysqlCLI" label="mysql CLI" default>

使用 mysql CLI 最简单的方式是从 StarRocks 容器 `starrocks-fe` 中运行它：

```bash
docker exec -it quickstart \
mysql -P 9030 -h 127.0.0.1 -u root --prompt="StarRocks > "
```

:::tip
所有 `docker compose` 命令必须在包含 `docker-compose.yml` 文件的目录中运行。
:::

如果您想要安装 mysql CLI，请展开下方的 **mysql client install** 查看详情：

<details>

<summary>mysql client install</summary>

- macOS：如果您使用Homebrew且不需要MySQL服务器，请运行 `brew install mysql` 来安装CLI。
- Linux：查看您的软件仓库系统中的 `mysql` 客户端。例如，`yum install mariadb`。
- Microsoft Windows：安装 [MySQL Community Server](https://dev.mysql.com/downloads/mysql/) 并运行所提供的客户端，或在WSL中运行 `mysql`。

</details>

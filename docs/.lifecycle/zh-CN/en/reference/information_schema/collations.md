# 整理

`collations` 包含可用的整理。

`collations` 中提供以下字段：

| **字段**           | **描述**                                                     |
| ------------------ | ------------------------------------------------------------ |
| COLLATION_NAME     | 整理的名称。                                                |
| CHARACTER_SET_NAME | 与整理关联的字符集的名称。                                  |
| ID                 | 整理的 ID。                                                 |
| IS_DEFAULT         | 整理是否为其字符集的默认值。                                |
| IS_COMPILED        | 字符集是否已编译到服务器中。                                 |
| SORTLEN            | 这与以字符集表示的字符串所需的排序内存量有关。               |

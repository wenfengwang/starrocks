# schemata

`schemata` 提供关于数据库的信息。

在 `schemata` 中提供以下字段：

| **字段**                    | **描述**                                              |
| -------------------------- | ------------------------------------------------------------ |
| CATALOG_NAME               | 模式所属目录的名称。该值始终为 `NULL`。 |
| SCHEMA_NAME                | 模式的名称。                                      |
| DEFAULT_CHARACTER_SET_NAME | 模式的默认字符集。                            |
| DEFAULT_COLLATION_NAME     | 模式的默认排序规则。                                |
| SQL_PATH                   | 该值始终为 `NULL`。                                 |

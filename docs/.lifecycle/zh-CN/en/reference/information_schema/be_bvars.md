# be_bvars

`be_bvars`提供有关bRPC的统计信息。您可以查看StarRocks的某些组件的RPC延迟、QPS和其他统计信息。

`be_bvars`提供以下字段：

| **字段**   | **描述**                                                     |
| --------- | ------------------------------------------------------------ |
| BE_ID     | bvar所在BE的ID。                                               |
| NAME     | bvar的名称。                                                  |
| DESC     | bvar的描述，包括重要的统计信息。                             |

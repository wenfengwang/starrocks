# 任务

`任务` 提供有关异步任务的信息。

在 `任务` 中提供了以下字段：

| **字段**      | **描述**                                                     |
| ----------- | ------------------------------------------------------------ |
| 任务名称       | 任务的名称。                                                   |
| 创建时间       | 任务创建的时间。                                                |
| 计划           | 任务计划。如果任务定期触发，该字段显示 `每个xxx开始xxx`。        |
| 数据库         | 任务所属的数据库。                                              |
| 定义           | 任务的 SQL 定义。                                              |
| 过期时间       | 任务过期的时间。                                                |

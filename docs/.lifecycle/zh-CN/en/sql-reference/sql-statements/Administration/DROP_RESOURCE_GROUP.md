# 删除资源组

## 描述

删除指定的资源组。

## 语法

```SQL
DROP RESOURCE GROUP <resource_group_name>
```

## 参数

| **参数**             | **描述**                               |
| ------------------- | ----------------------------------------- |
| resource_group_name | 要删除的资源组的名称。 |

## 例子

例子 1：删除资源组 `rg1`。

```SQL
DROP RESOURCE GROUP rg1;
```

# 年份

## 描述

返回日期中的年份部分，并返回一个从1000到9999范围内的值。

`date`参数必须是DATE或DATETIME类型。

## 语法

```Haskell
INT YEAR(DATETIME date)
```

## 示例

```Plain Text
MySQL > select year('1987-01-01');
+-----------------------------+
| year('1987-01-01 00:00:00') |
+-----------------------------+
|                        1987 |
+-----------------------------+
```

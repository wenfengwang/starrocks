# from_days

## 描述

返回从0000-01-01开始的日期。

## 语法

```Haskell
DATE FROM_DAYS(INT N)
```

## 例子

```Plain Text
MySQL > select from_days(730669);
+-------------------+
| from_days(730669) |
+-------------------+
| 2000-07-03        |
+-------------------+
```

## 关键字

FROM_DAYS,FROM,DAYS

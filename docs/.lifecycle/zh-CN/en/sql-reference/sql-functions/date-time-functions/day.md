# day

## 描述

从日期或日期时间表达式中提取天数部分，并返回一个范围从1到31的值。

`date` 参数必须是 DATE 或 DATETIME 类型。

## 语法

```Haskell
INT DAY(DATETIME|DATE date)
```

## 示例

```Plain Text
MySQL > select day('1987-01-31');
+----------------------------+
| day('1987-01-31 00:00:00') |
+----------------------------+
|                         31 |
+----------------------------+

MySQL > select day('1987-01-31 20:10:59');
+----------------------------+
| day('1987-01-31 20:10:59') |
+----------------------------+
|                         31 |
+----------------------------+
```

# pmod

## 描述

返回 `被除数` 除以 `除数` 的正余数。

## 语法

```SQL
pmod(被除数, 除数)
```

## 参数

- `被除数`：要被除的数。
- `除数`：用来除的数。

`被除数` 和 `除数` 都支持以下数据类型：

- BIGINT
- DOUBLE

> **注意**
>
> `被除数` 和 `除数` 必须在数据类型上达成一致。如果它们的数据类型不一致，StarRocks 将执行隐式转换。

## 返回值

返回与 `被除数` 相同数据类型的值。如果指定 `除数` 为0，StarRocks 将返回NULL。

## 示例

```Plain
mysql> select pmod(3.14,3.14);
+------------------+
| pmod(3.14, 3.14) |
+------------------+
|                0 |
+------------------+

mysql> select pmod(3,6);
+------------+
| pmod(3, 6) |
+------------+
|          3 |
+------------+

mysql> select pmod(11,5);
+-------------+
| pmod(11, 5) |
+-------------+
|           1 |
+-------------+

mysql> select pmod(-11,5);
+--------------+
| pmod(-11, 5) |
+--------------+
|            4 |
+--------------+

mysql> SELECT pmod(11,-5);
+--------------+
| pmod(11, -5) |
+--------------+
|           -4 |
+--------------+

mysql> SELECT pmod(-11,-5);
+---------------+
| pmod(-11, -5) |
+---------------+
|            -1 |
+---------------+
```

# log10, dlog10

## 描述

计算一个数的以10为底的对数。

## 语法

```SQL
log10(arg)
```

## 参数

- `arg`：要计算对数的值。仅支持DOUBLE数据类型。

> **注意**
>
> 如果指定的 `arg` 为负数或0，StarRocks 返回 NULL。

## 返回值

返回DOUBLE数据类型的值。

## 示例

例1：计算100的以10为底的对数。

```Plain
select log10(100);
+------------+
| log10(100) |
+------------+
|          2 |
+------------+
1 行 (0.02 秒)
```

# mod

## 描述

返回`被除数`除以`除数`的余数的模运算函数。

## 语法

```SQL
mod(被除数, 除数)
```

## 参数

- `被除数`：要被除的数。
- `除数`：用来除的数。

`被除数`和`除数`都支持以下数据类型：

- TINYINT
- SMALLINT
- INT
- BIGINT
- LARGEINT
- FLOAT
- DOUBLE
- DECIMALV2
- DECIMAL32
- DECIMAL64
- DECIMAL128

> **注意**
>
> 如果`被除数`和`除数`的数据类型不相符，StarRocks会执行隐式转换。

## 返回值

返回值与`被除数`相同的数据类型。如果指定的`除数`为0，StarRocks返回NULL。

## 示例

```Plain
mysql> select mod(3.14,3.14);
+-----------------+
| mod(3.14, 3.14) |
+-----------------+
|               0 |
+-----------------+

mysql> select mod(3.14, 3);
+--------------+
| mod(3.14, 3) |
+--------------+
|         0.14 |
+--------------+

select mod(11,-5);
+------------+
| mod(11, -5)|
+------------+
|          1 |
+------------+

select mod(-11,5);
+-------------+
| mod(-11, 5) |
+-------------+
|          -1 |
+-------------+
```

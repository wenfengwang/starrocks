# cos

## 描述

计算参数的余弦值。

## 语法

```Haskell
DOUBLE cos(DOUBLE arg)
```

### 参数

`arg`: 您只能指定一个数值。此函数在计算该值的余弦值之前将数值转换为DOUBLE值。

## 返回值

返回DOUBLE数据类型的值。

## 使用注意事项

如果您指定一个非数值，此函数将返回`NULL`。

## 示例

```Plain
mysql> select cos(-1);
+--------------------+
| cos(-1)            |
+--------------------+
| 0.5403023058681398 |
+--------------------+

mysql> select cos(0);
+--------+
| cos(0) |
+--------+
|      1 |
+--------+

mysql> select cos(1);
+--------------------+
| cos(1)             |
+--------------------+
| 0.5403023058681398 |
+--------------------+

mysql> select cos("");
+---------+
| cos('') |
+---------+
|    NULL |
+---------+
```

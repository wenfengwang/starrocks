# exp,dexp

## 描述

返回e的`x`次幂的值。此函数称为自然对数函数。

## 语法

```SQL
EXP(x);
```

## 参数

`x`: 幂的数值。支持DOUBLE类型。

## 返回值

返回DOUBLE数据类型的值。

## 示例

返回e的3.14次幂：

```Plaintext
mysql> select exp(3.14);
+--------------------+
| exp(3.14)          |
+--------------------+
| 23.103866858722185 |
+--------------------+
1 row in set (0.01 sec)
```

# 双曲正切（TANH）

## 描述

计算参数的双曲正切值。

此函数支持v3.0及更高版本。

## 语法

```Haskell
DOUBLE tanh(DOUBLE arg)
```

### 参数

`arg`：您只能指定一个数值。在计算数值的双曲正切值之前，此函数将数值转换为DOUBLE值。

## 返回值

返回DOUBLE数据类型的值。

## 使用注意事项

如果指定了非数值参数，此函数将返回 `NULL`。

## 示例

```Plain
mysql> select tanh(-1);
+---------------------+
| tanh(-1)            |
+---------------------+
| -0.7615941559557649 |
+---------------------+

mysql> select tanh(0);
+---------+
| tanh(0) |
+---------+
|       0 |
+---------+

mysql> select tanh(1);
+--------------------+
| tanh(1)            |
+--------------------+
| 0.7615941559557649 |
+--------------------+

mysql> select tanh("");
+----------+
| tanh('') |
+----------+
|     NULL |
+----------+
```

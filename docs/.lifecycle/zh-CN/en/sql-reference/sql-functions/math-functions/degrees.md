# 度

## 描述

将弧度 `x` 转换为度数。

## 语法

```SQL
DEGREES(x);
```

## 参数

`x`: 弧度。支持DOUBLE数据类型。

## 返回值

返回DOUBLE数据类型的值。

## 示例

```Plaintext
mysql> select degrees(3.1415926535898);
+--------------------------+
| degrees(3.1415926535898) |
+--------------------------+
|        180.0000000000004 |
+--------------------------+
1 row in set (0.07 sec)
```

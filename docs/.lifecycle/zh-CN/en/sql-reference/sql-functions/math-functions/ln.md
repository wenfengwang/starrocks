# ln, dlog1, log

## 描述

计算一个数的自然对数（以`e`为底）。

## 语法

```SQL
ln(arg)
log(arg)
dlog1(arg)
```

## 参数

- `arg`: 想要计算其对数的值。仅支持DOUBLE数据类型。

> **注意**
>
> 如果指定的`arg`为负数或0，StarRocks返回NULL。

## 返回值

返回DOUBLE数据类型的值。

## 示例

示例1：计算3的自然对数。

```SQL
mysql> select ln(3);
+--------------------+
| ln(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

mysql> select dlog1(3);
+--------------------+
| dlog1(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

mysql> select log(3);
+--------------------+
| log(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

```

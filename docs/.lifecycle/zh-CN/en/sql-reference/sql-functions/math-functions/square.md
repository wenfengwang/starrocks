# 平方

## 描述

计算一个值的平方。

## 语法

```Haskell
square(arg)
```

### 参数

`arg`：要计算其平方的值。您只能指定一个数值。此函数在计算该值的平方之前将数值转换为DOUBLE值。

## 返回值

返回DOUBLE数据类型的值。

## 使用说明

如果指定了非数值的值，则该函数返回 `NULL`。

## 示例

示例1：计算一个数值的平方。

```Plain
mysql>  select square(11);
+------------+
| square(11) |
+------------+
|        121 |
+------------+
```

示例2：计算一个非数值的值的平方。返回值为 `NULL`。

```Plain
mysql>  select square('2021-01-01');
+----------------------+
| square('2021-01-01') |
+----------------------+
|                 NULL |
+----------------------+
```

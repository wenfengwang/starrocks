# e

## 描述

返回常数 e（自然对数的底）。

## 语法

```SQL
E();
```

## 参数

此函数不接受任何参数。

## 返回值

以DOUBLE形式返回常数 e。

## 示例

```Plaintext
mysql> select e();
+-------------------+
| e()               |
+-------------------+
| 2.718281828459045 |
+-------------------+
1 行受影响 (0.01 秒)
```

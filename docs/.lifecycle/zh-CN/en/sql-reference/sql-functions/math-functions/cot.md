# cot

## 描述

返回以弧度为单位的角 `x` 的余切值。如果弧度为 `x` 的角度为 `0` 或 `NULL`，则返回 `NULL`。

## **语法**

```SQL
COT(x);
```

## **参数**

`x`：以弧度表示的角度。支持DOUBLE类型。

## **返回值**

返回DOUBLE数据类型的值。

## **示例**

```Plaintext
mysql> select cot(3.1415926/2);
+------------------------+
| cot(3.1415926 / 2)     |
+------------------------+
| 2.6794896585028646e-08 |
+------------------------+
1 行受影响 (0.25 秒)

mysql> select cot(0);
+--------+
| cot(0) |
+--------+
|   NULL |
+--------+
1 行受影响 (0.03 秒)
```

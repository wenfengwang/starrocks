# acos

## 描述

计算参数的反余弦。

## 语法

```Haskell
DOUBLE acos(DOUBLE arg)
```

### 参数

`arg`: 您只能指定一个数值。此函数在计算值的反余弦之前将数值转换为DOUBLE值。

## 返回值

返回DOUBLE数据类型的值。

## 使用注意事项

如果指定了非数值，该函数将返回`NULL`。

## 示例

```Plain
mysql> select acos(-1);
+-------------------+
| acos(-1)          |
+-------------------+
| 3.141592653589793 |
+-------------------+

mysql> select acos(0);
+--------------------+
| acos(0)            |
+--------------------+
| 1.5707963267948966 |
+--------------------+

mysql> select acos(1);
+---------+
| acos(1) |
+---------+
|       0 |
+---------+

mysql> select acos("");
+----------+
| acos('') |
+----------+
|     NULL |
+----------+
```

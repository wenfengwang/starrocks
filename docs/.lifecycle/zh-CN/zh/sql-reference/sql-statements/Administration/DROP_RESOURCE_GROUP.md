# 删除资源组

## 功能

删除指定的资源组。

## 语法

```SQL
DROP RESOURCE GROUP <resource_group_name>
```

## 参数说明

| **参数**            | **说明**         |
| ------------------- | ---------------- |
| resource_group_name | 待删除的资源组名称。 |

## 示例

示例一：删除资源组 `rg1`。

```SQL
DROP RESOURCE GROUP rg1;
```

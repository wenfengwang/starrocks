# クイックスタート

これらのクイックスタートガイドは、小規模なStarRocks環境での作業を始める助けになるでしょう。あなたが立ち上げるクラスターは、StarRocksの動作を学ぶのに適していますが、大規模なデータセットの分析やパフォーマンステストを行うためには意図されていません。

スケーラブルなデプロイメントについては、[StarRocksのデプロイ](../deployment/deployment_overview.md) をご覧ください。

# クイックスタート

これらのクイックスタートガイドは、小規模なStarRocks環境での作業を助けます。起動するクラスターは、StarRocksの動作を学習するのに適していますが、大規模なデータセットの解析やパフォーマンステストには適していません。

スケーラブルな展開については、[StarRocksの展開](../deployment/deployment_overview.md)を参照してください。

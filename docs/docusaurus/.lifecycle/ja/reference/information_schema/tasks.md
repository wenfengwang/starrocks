# タスク

`タスク`は非同期タスクに関する情報を提供します。

`タスク`には次のフィールドがあります:

| **フィールド** | **説明**                                                    |
| ----------- | ------------------------------------------------------------ |
| TASK_NAME   | タスクの名前です。                                           |
| CREATE_TIME | タスクが作成された時間です。                                |
| SCHEDULE    | タスクのスケジュールです。定期的にトリガーされる場合、このフィールドには `START xxx EVERY xxx` が表示されます。 |
| DATABASE    | タスクが属するデータベースです。                             |
| DEFINITION  | タスクのSQL定義です。                                       |
| EXPIRE_TIME | タスクの有効期限です。                                       |

データベースを作成

これらの2行を `StarRocks > ` プロンプトで入力し、それぞれ入力後にエンターキーを押してください：

```sql
CREATE DATABASE IF NOT EXISTS quickstart;

USE quickstart;
```

2つのテーブルを作成

#### Crashdata

クラッシュデータセットには、これよりも多くのフィールドが含まれています。スキーマは、運転条件に関する質問に回答するのに役立つフィールドのみを含むように整理されています。

```sql
CREATE TABLE IF NOT EXISTS crashdata (
    CRASH_DATE DATETIME,
    BOROUGH STRING,
    ZIP_CODE STRING,
    LATITUDE INT,
    LONGITUDE INT,
    LOCATION STRING,
    ON_STREET_NAME STRING,
    CROSS_STREET_NAME STRING,
    OFF_STREET_NAME STRING,
    CONTRIBUTING_FACTOR_VEHICLE_1 STRING,
    CONTRIBUTING_FACTOR_VEHICLE_2 STRING,
    COLLISION_ID INT,
    VEHICLE_TYPE_CODE_1 STRING,
    VEHICLE_TYPE_CODE_2 STRING
);
```

#### Weatherdata

クラッシュデータと同様に、天候データセットには多くの列（合計125列）があり、質問に回答するために予想される列のみがデータベースに含まれています。

```sql
CREATE TABLE IF NOT EXISTS weatherdata (
    DATE DATETIME,
    NAME STRING,
    HourlyDewPointTemperature STRING,
    HourlyDryBulbTemperature STRING,
    HourlyPrecipitation STRING,
    HourlyPresentWeatherType STRING,
    HourlyPressureChange STRING,
    HourlyPressureTendency STRING,
    HourlyRelativeHumidity STRING,
    HourlySkyConditions STRING,
    HourlyVisibility STRING,
    HourlyWetBulbTemperature STRING,
    HourlyWindDirection STRING,
    HourlyWindGustSpeed STRING,
    HourlyWindSpeed STRING
);
```

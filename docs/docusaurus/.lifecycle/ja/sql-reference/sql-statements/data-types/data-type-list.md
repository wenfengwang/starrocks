# データタイプの概要

StarRocksは以下のデータタイプをサポートしています：[数値](#numeric)、[文字列](#string)、[日付](#date)、[セミストラクチャード](#semi-structured)、そして[その他](#others)。テーブルの作成時にこれらのタイプのカラムを定義したり、これらのカラムにデータをロードしたり、データのクエリを実行することができます。

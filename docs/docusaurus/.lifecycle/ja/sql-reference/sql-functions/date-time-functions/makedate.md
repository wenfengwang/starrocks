# makedate

## 説明

指定された年と年間日数の値に基づいて日付を作成および返します。

この機能はv3.1からサポートされています。

## 構文

```Haskell
DATE makedate(INT year, INT dayOfYear);
```

## パラメーター

- `year`: 0から9999までの範囲です。この範囲を超えるとNULLが返されます。サポートされているデータ型はINTです。
- `dayOfYear`: 年間日数です。サポートされているデータ型はINTです。この数値が366を超えたり、閏年でない年の366の場合、年日数ではありません（[dayofyear](./dayofyear.md)関数と同じ意味を維持するため）。

## 戻り値

指定された年のdayOfYear番目の日付が返されます。

- `year`は[0,9999]の範囲内でなければなりません。それ以外の場合、NULLが返されます。
- `dayOfYear`は現在の年の日数（通常の年は365日、閏年は366日）の範囲内でなければなりません。それ以外の場合、NULLが返されます。
- どちらかの入力パラメーターがNULLの場合、結果もNULLです。

## 例

```Plain Text
mysql> select makedate(2023,0);
+-------------------+
| makedate(2023, 0) |
+-------------------+
| NULL              |
+-------------------+

mysql> select makedate(2023,32);
+--------------------+
| makedate(2023, 32) |
+--------------------+
| 2023-02-01         |
+--------------------+

mysql> select makedate(2023,365);
+---------------------+
| makedate(2023, 365) |
+---------------------+
| 2023-12-31          |
+---------------------+

mysql> select makedate(2023,366);
+---------------------+
| makedate(2023, 366) |
+---------------------+
| NULL                |
+---------------------+

mysql> select makedate(9999,365);
+---------------------+
| makedate(9999, 365) |
+---------------------+
| 9999-12-31          |
+---------------------+

mysql> select makedate(10000,1);
+--------------------+
| makedate(10000, 1) |
+--------------------+
| NULL               |
+--------------------+
```

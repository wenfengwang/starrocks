# 月

## 説明

指定された日付の月を返します。返される値の範囲は1から12です。

`date`パラメータはDATEまたはDATETIMEタイプである必要があります。

## 構文

```Haskell
INT MONTH(DATETIME date)
```

## 例

```Plain Text
MySQL > select month('1987-01-01');
+-----------------------------+
|month('1987-01-01 00:00:00') |
+-----------------------------+
|                           1 |
+-----------------------------+
```

## キーワード

MONTH

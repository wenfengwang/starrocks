# date_format

## 説明

指定された形式に従って、日付を文字列に変換します。 現在、最大128バイトの文字列をサポートしています。 返される値の長さが128を超える場合、NULLが返されます。

## 構文

```Haskell
VARCHAR DATE_FORMAT(DATETIME date, VARCHAR format)
```

## パラメータ

- 「date」パラメータは有効な日付または日付式である必要があります。

- 「format」は日付または時間の出力形式を指定します。

利用可能な形式は以下のとおりです。

```Plain Text
%a | 省略された曜日名（日曜日から土曜日まで）
%b | 省略された月の名前（JanからDec）
%c | 月の名前の数値形式（0〜12）
%D | 英語の接尾辞に続く、数値値としての月の日
%d | 数値値としての月の日（00〜31）
%e | 数値値としての月の日（0〜31）
%f | マイクロ秒
%H | 時間（00〜23）
%h | 時間（01〜12）
%I | 時間（01〜12）
%i | 分（00〜59）
%j | 年の日（001〜366）
%k | 時間（0〜23）
%l | 時間（1〜12）
%M | 月の完全な名前
%m | 月の数値形式（00〜12）
%p | 午前または午後
%r | 12時間制の時刻（hh:mm:ss AMまたはPM）
%S | 秒（00〜59）
%s | 秒（00〜59）
%T | 24時間形式の時刻（hh:mm:ss）
%U | 週（00〜53、日曜日が週の最初の日の場合）
%u | 週（00〜53、月曜日が週の最初の日の場合）
%V | 週（01〜53、日曜日が週の最初の日の場合。 %Xと組み合わせて使用します）
%v | 週（01〜53、月曜日が週の最初の日の場合。 %xと組み合わせて使用します）
%W | 曜日の完全な名前
%w | 日曜日=0、土曜日=6での曜日
%X | 日曜日が週の最初の日の年。4桁の値。 %Vと組み合わせて使用します。
%x | 月曜日が週の最初の日の年。4桁の値。 %vと組み合わせて使用します。
%Y | 年。4桁の値。
%y | 年。2桁の値。
%% |   %を表す。
```

## 例

```Plain Text
MySQL > select date_format('2009-10-04 22:23:00', '%W %M %Y');
+------------------------------------------------+
| date_format('2009-10-04 22:23:00', '%W %M %Y') |
+------------------------------------------------+
| Sunday October 2009                            |
+------------------------------------------------+

MySQL > select date_format('2007-10-04 22:23:00', '%H:%i:%s');
+------------------------------------------------+
| date_format('2007-10-04 22:23:00', '%H:%i:%s') |
+------------------------------------------------+
| 22:23:00                                       |
+------------------------------------------------+

MySQL > select date_format('1900-10-04 22:23:00', '%D %y %a %d %m %b %j');
+------------------------------------------------------------+
| date_format('1900-10-04 22:23:00', '%D %y %a %d %m %b %j') |
+------------------------------------------------------------+
| 4th 00 Thu 04 10 Oct 277                                   |
+------------------------------------------------------------+

MySQL > select date_format('1997-10-04 22:23:00', '%H %k %I %r %T %S %w');
+------------------------------------------------------------+
| date_format('1997-10-04 22:23:00', '%H %k %I %r %T %S %w') |
+------------------------------------------------------------+
| 22 22 10 10:23:00 PM 22:23:00 00 6                         |
+------------------------------------------------------------+

MySQL > select date_format('1999-01-01 00:00:00', '%X %V');
+---------------------------------------------+
| date_format('1999-01-01 00:00:00', '%X %V') |
+---------------------------------------------+
| 1998 52                                     |
+---------------------------------------------+

MySQL > select date_format('2006-06-01', '%d');
+------------------------------------------+
| date_format('2006-06-01 00:00:00', '%d') |
+------------------------------------------+
| 01                                       |
+------------------------------------------+

MySQL > select date_format('2006-06-01', '%%%d');
+--------------------------------------------+
| date_format('2006-06-01 00:00:00', '%%%d') |
+--------------------------------------------+
| %01                                        |
+--------------------------------------------+
```

## キーワード

DATE_FORMAT,DATE,FORMAT

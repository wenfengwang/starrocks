# years_sub

## 説明

指定された日時または日付から指定された年数を減算します。

## 構文

```Haskell
DATETIME YEARS_SUB(DATETIME date, INT years)
```

## パラメータ

`date`: 元の日時、DATETIMEまたはDATEの型。


`years`: 減算する年数。値は負であってもよいが、日付の年から年を引いた値は10000を超えてはならない。例えば、日付の年が2022年である場合、yearsは-7979より小さくてはならない。同時に、年は日付の年の値を超えてはならない。例えば、日付の年の値が2022年である場合、yearsは2022より大きくてはならない。


## 戻り値

戻り値の型はパラメータ`date`と同じです。結果の年が[0、9999]の範囲外の場合は、NULLを返します。

## 例

```Plain Text
select years_sub("2022-12-20 15:50:21", 2);
+-------------------------------------+
| years_sub('2022-12-20 15:50:21', 2) |
+-------------------------------------+
| 2020-12-20 15:50:21                 |
+-------------------------------------+
```

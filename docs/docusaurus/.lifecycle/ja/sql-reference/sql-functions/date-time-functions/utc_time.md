# utc_time

## Description

現在のUTC時間を 'HH:MM:SS' の値として返します。

## Syntax

```Haskell
TIME UTC_TIME()
```

## Examples

```Plain Text
MySQL > select utc_time();
+----------------+
| utc_time()     |
+----------------+
| 12:31:18       |
+----------------+
```

## keyword

UTC_TIME,UTC,TIME

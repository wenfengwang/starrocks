# array_remove（配列削除）

## 説明

配列から要素を削除します。

## 構文

```Haskell
array_remove(任意の配列, 任意の要素)
```

## パラメーター

- `任意の配列`: 検索される配列です。
- `任意の要素`: 配列内の要素に一致する式です。

## 戻り値

指定された要素が削除された配列を返します。

## 例

```plaintext
mysql> select array_remove([1,2,3,null,3], 3);

+---------------------------------+

| array_remove([1,2,3,NULL,3], 3) |

+---------------------------------+

| [1,2,null]                      |

+---------------------------------+

1 行が返されました（0.01 秒）
```

## キーワード

ARRAY_REMOVE, ARRAY

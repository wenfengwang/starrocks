# e（自然対数の底）

## 説明

自然対数の底である定数 e を返します。

## 構文

```SQL
E();
```

## パラメータ

この関数にはパラメータはありません。

## 戻り値

定数eをDOUBLE型で返します。

## 例

```Plaintext
mysql> select e();
+-------------------+
| e()               |
+-------------------+
| 2.718281828459045 |
+-------------------+
1行 in set (0.01 sec)
```

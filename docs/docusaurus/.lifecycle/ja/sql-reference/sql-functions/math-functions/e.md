# e

## 描述

自然対数の底である定数 e を返します。

## 構文

```SQL
E();
```

## パラメータ

この関数にはパラメータはありません。

## 戻り値

DOUBLE 型として定数 e を返します。

## 例

```Plaintext
mysql> select e();
+-------------------+
| e()               |
+-------------------+
| 2.718281828459045 |
+-------------------+
1 行が返されました (0.01 秒)
```

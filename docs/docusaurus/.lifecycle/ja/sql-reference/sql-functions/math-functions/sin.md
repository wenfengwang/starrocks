# sin

## 説明

`x`がラジアン単位の場合、`x`の正弦を返します。

## 構文

```Haskell
SIN(x);
```

## パラメータ

`x`: DOUBLEデータ型をサポートしています。

## 戻り値

DOUBLEデータ型の値を返します。

## 例

```Plain
mysql> select sin(3.14);
+-----------------------+
| sin(3.14)             |
+-----------------------+
| 0.0015926529164868282 |
+-----------------------+
1 row in set (0.21 sec)
```

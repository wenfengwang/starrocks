# radians

## 説明

`x`を角度からラジアンに変換します。

## 構文

```Haskell
REDIANS(x);
```

## パラメーター

`x`: DOUBLEデータ型をサポートしています。

## 戻り値

DOUBLEデータ型の値を返します。

## 例

```Plain
mysql> select radians(90);
+--------------------+
| radians(90)        |
+--------------------+
| 1.5707963267948966 |
+--------------------+
1行がセットされました (0.00秒)
```

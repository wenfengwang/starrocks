# exp,dexp

## 説明

`x`の累乗のeの値を返します。この関数は自然対数関数と呼ばれます。

## 構文

```SQL
EXP(x);
```

## パラメータ

`x`: 累乗の数。DOUBLE がサポートされています。

## 戻り値

DOUBLEデータ型の値を返します。

## 例

3.14のeの累乗を返す：

```Plaintext
mysql> select exp(3.14);
+--------------------+
| exp(3.14)          |
+--------------------+
| 23.103866858722185 |
+--------------------+
1 row in set (0.01 sec)
```

# cot（コタンジェント）

## 説明

ラジアンでの角度`x`のコタジェントを返します。角度`x`が`0`または`NULL`の場合、`NULL`が返されます。

## **構文**

```SQL
COT(x);
```

## **パラメーター**

`x`: ラジアンでの角度。DOUBLEがサポートされています。

## **戻り値**

DOUBLEデータ型の値を返します。

## **例**

```Plaintext
mysql> select cot(3.1415926/2);
+------------------------+
| cot(3.1415926 / 2)     |
+------------------------+
| 2.6794896585028646e-08 |
+------------------------+
1 row in set (0.25 sec)

mysql> select cot(0);
+--------+
| cot(0) |
+--------+
|   NULL |
+--------+
1 row in set (0.03 sec)
```

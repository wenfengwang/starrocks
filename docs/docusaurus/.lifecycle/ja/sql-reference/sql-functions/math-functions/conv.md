# conv

## 説明

数値 `x` を一つの数値基数系から別のものに変換し、その結果を文字列値として返します。

## 構文

```Haskell
CONV(x,y,z);
```

## パラメータ

- `x`: 変換対象の数値。VARCHAR または BIGINT がサポートされています。
- `y`: 元の基数。TINYINT がサポートされています。
- `z`: ターゲット基数。TINYINT がサポートされています。

## 戻り値

VARCHAR データ型の値を返します。

## 例

数値 8 を数値基数系2から数値基数系10に変換する場合:

```Plain
mysql> select conv(8,10,2);
+----------------+
| conv(8, 10, 2) |
+----------------+
| 1000           |
+----------------+
1 行が返されました (0.00 秒)
```

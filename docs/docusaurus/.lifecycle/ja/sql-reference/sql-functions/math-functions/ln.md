# ln, dlog1, log

## 説明

数値の自然対数（`e`を底とする対数）を計算します。

## 構文

```SQL
ln(arg)
log(arg)
dlog1(arg)
```

## パラメータ

- `arg`: 対数を計算したい値。DOUBLEデータ型のみがサポートされています。

> **注記**
>
> `arg`が負または0と指定されている場合、StarRocksはNULLを返します。

## 戻り値

DOUBLEデータ型の値が返されます。

## 例

例1: 3の自然対数を計算する。

```SQL
mysql> select ln(3);
+--------------------+
| ln(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

mysql> select dlog1(3);
+--------------------+
| dlog1(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)

mysql> select log(3);
+--------------------+
| log(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 row in set (0.00 sec)
```

# log10, dlog10

## 説明

数値の底10の対数を計算します。

## 構文

```SQL
log10(arg)
```

## パラメーター

- `arg`: 対数を計算したい値です。DOUBLEデータ型のみをサポートしています。

> **注記**
>
> StarRocksは、`arg`が負の数または0として指定された場合、NULLを返します。

## 戻り値

DOUBLEデータ型の値を返します。

## 例

例1：100の底10の対数を計算します。

```Plain
select log10(100);
+------------+
| log10(100) |
+------------+
|          2 |
+------------+
1行がセットにあります (0.02秒)
```

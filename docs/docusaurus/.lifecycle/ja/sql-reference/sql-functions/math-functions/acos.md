# acos

## 説明

引数の逆余弦を計算します。

## 構文

```Haskell
DOUBLE acos(DOUBLE arg)
```

### パラメータ

`arg`: 数値のみを指定できます。この関数は、値の逆余弦を計算する前に数値をDOUBLE値に変換します。

## 戻り値

DOUBLEデータ型の値を返します。

## 使用上の注意

数値以外の値を指定した場合、この関数は`NULL`を返します。

## 例

```Plain
mysql> select acos(-1);
+-------------------+
| acos(-1)          |
+-------------------+
| 3.141592653589793 |
+-------------------+

mysql> select acos(0);
+--------------------+
| acos(0)            |
+--------------------+
| 1.5707963267948966 |
+--------------------+

mysql> select acos(1);
+---------+
| acos(1) |
+---------+
|       0 |
+---------+

mysql> select acos("");
+----------+
| acos('') |
+----------+
|     NULL |
+----------+
```

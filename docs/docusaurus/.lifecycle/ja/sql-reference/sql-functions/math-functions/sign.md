# sign（サイン）

## 説明

`x`のサインを返します。入力として負の数、0、または正の数がある場合、それぞれ出力は`-1`、`0`、または`1`に対応します。

## 構文

```Haskell
SIGN(x);
```

## パラメーター

`x`: DOUBLEデータ型をサポートしています。

## 戻り値

FLOATデータ型の値を返します。

## 例

```Plain
mysql> select sign(3.14159);
+---------------+
| sign(3.14159) |
+---------------+
|             1 |
+---------------+
1行が返されました (0.02秒)
```

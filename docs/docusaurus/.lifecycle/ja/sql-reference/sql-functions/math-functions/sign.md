# sign（符号）

## 説明

`x` の符号を返します。負の数、0、または正の数が入力として与えられた場合、それぞれに対応する出力は `-1`、`0`、または `1` になります。

## 構文

```Haskell
SIGN(x);
```

## パラメータ

`x`: DOUBLE データ型をサポートしています。

## 戻り値

FLOAT データ型の値を返します。

## 例

```Plain
mysql> select sign(3.14159);
+---------------+
| sign(3.14159) |
+---------------+
|             1 |
+---------------+
1 行が返されました (0.02 秒)
```

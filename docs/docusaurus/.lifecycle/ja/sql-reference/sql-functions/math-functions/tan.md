# tan（タンジェント）

## 説明

`x` がラジアンである場合、`x` のタンジェントを返します。

## 構文

```Haskell
TAN(x);
```

## パラメーター

`x`: DOUBLE データ型をサポートしています。

## 戻り値

DOUBLE データ型の値を返します。

## 例

```Plain
mysql> select tan(3.14);
+-----------------------+
| tan(3.14)             |
+-----------------------+
| -0.001592654936407223 |
+-----------------------+
1 行が返されました (0.12 秒)
```

# length

## Description

この関数は、文字列の長さ（バイト単位）を返します。

## Syntax

```Haskell
INT length(VARCHAR str)
```

## Examples

```Plain Text
MySQL > select length("abc");
+---------------+
| length('abc') |
+---------------+
|             3 |
+---------------+
```

## keyword

LENGTH

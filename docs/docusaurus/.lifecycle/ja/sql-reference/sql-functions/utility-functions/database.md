# データベース

## 説明

現在のデータベースの名前を返します。データベースが選択されていない場合、空の値が返されます。

## 構文

```Haskell
database()
```

## パラメーター

この関数にはパラメーターは必要ありません。

## 戻り値

現在のデータベースの名前を文字列で返します。

## 例

```sql
-- 宛先データベースを選択する。
use db_test

-- 現在のデータベースの名前をクエリする。
select database();
+------------+
| DATABASE() |
+------------+
| db_test    |
+------------+
```

## 関連項目

[USE](../../sql-statements/data-definition/USE.md): 宛先データベースに切り替えます。

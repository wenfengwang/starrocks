# 切り上げ

## 説明

入力の `arg` を最も近い等しいか、それより大きい整数に丸めた値を返します。

## 構文

```Shell
ceiling(arg)
```

## パラメータ

`arg` は DOUBLE データ型をサポートしています。

## 戻り値

BIGINT データ型の値を返します。

## 例

```Plain
mysql> select ceiling(3.14);
+---------------+
| ceiling(3.14) |
+---------------+
|             4 |
+---------------+
1 行が返されました (0.00 秒)
```

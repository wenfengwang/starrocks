# e

## 説明

自然対数の底である定数eを返します。

## 構文

```SQL
E();
```

## パラメーター

この関数にはパラメーターはありません。

## 戻り値

定数eをDOUBLE型で返します。

## 例

```Plaintext
mysql> select e();
+-------------------+
| e()               |
+-------------------+
| 2.718281828459045 |
+-------------------+
1 行が返されました (0.01 秒)
```

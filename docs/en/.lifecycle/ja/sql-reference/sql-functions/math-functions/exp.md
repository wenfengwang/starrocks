# exp,dexp

## 説明

`x`のべき乗の値を返します。この関数は自然対数関数と呼ばれます。

## 構文

```SQL
EXP(x);
```

## パラメーター

`x`: べき乗の数。DOUBLEがサポートされています。

## 戻り値

DOUBLEデータ型の値を返します。

## 例

3.14のべき乗のeを返します:

```Plaintext
mysql> select exp(3.14);
+--------------------+
| exp(3.14)          |
+--------------------+
| 23.103866858722185 |
+--------------------+
1 row in set (0.01 sec)
```

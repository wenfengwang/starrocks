# log10, dlog10

## 説明

数値の底10の対数を計算します。

## 構文

```SQL
log10(arg)
```

## パラメータ

- `arg`: 対数を計算したい値。DOUBLEデータ型のみサポートされています。

> **注意**
>
> `arg`が負数または0として指定された場合、StarRocksはNULLを返します。

## 返り値

DOUBLEデータ型の値を返します。

## 例

例1: 100の底10の対数を計算する。

```Plain
select log10(100);
+------------+
| log10(100) |
+------------+
|          2 |
+------------+
1行が返されました (0.02 sec)
```

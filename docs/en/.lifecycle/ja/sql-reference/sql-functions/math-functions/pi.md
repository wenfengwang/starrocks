# pi

## 説明

Piの値を返します。

## 構文

```Plain
pi()
```

## パラメーター

なし

## 戻り値

DOUBLE型のPiの値を返します。

## 例

```Plain
mysql> select pi();
+-------------------+
| pi()              |
+-------------------+
| 3.141592653589793 |
+-------------------+
1 行が返されました (0.00 秒)
```

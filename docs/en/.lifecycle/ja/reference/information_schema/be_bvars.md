# be_bvars

`be_bvars`はbRPCに関する統計情報を提供します。StarRocksの一部のコンポーネントのRPCレイテンシ、QPS、およびその他の統計情報を表示することができます。

`be_bvars`には以下のフィールドが提供されます：

| **フィールド** | **説明**                                                     |
| -------------- | ------------------------------------------------------------ |
| BE_ID          | bvarが配置されているBEのID。                                  |
| NAME           | bvarの名前。                                                  |
| DESC           | bvarの説明。重要な統計情報も含まれます。                       |

# クイックスタート

これらのクイックスタートガイドは、小規模なStarRocks環境での作業をサポートします。起動するクラスタは、StarRocksの動作を学ぶために適していますが、大規模なデータセットの分析やパフォーマンステストには適していません。

スケーラブルな展開については、[StarRocksの展開](../deployment/deployment_overview.md)を参照してください。

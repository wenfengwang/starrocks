#### 纽约市每小时有多少起车祸？

```sql
SELECT COUNT(*),
       date_trunc("hour", crashdata.CRASH_DATE) AS 时间
FROM crashdata
GROUP BY 时间
ORDER BY 时间 ASC
LIMIT 200;
```

#### 能见度差时在纽约市开车安全吗？

让我们看看能见度差（0到1.0英里之间）时的撞车次数。为了回答这个问题，在DATETIME列上跨越两个表使用JOIN。

```sql
SELECT COUNT(DISTINCT c.COLLISION_ID) AS 车祸数,
       truncate(avg(w.HourlyDryBulbTemperature), 1) AS 温度_F,
       truncate(avg(w.HourlyVisibility), 2) AS 能见度,
       max(w.HourlyPrecipitation) AS 降水量,
       date_format((date_trunc("hour", c.CRASH_DATE)), '%d %b %Y %H:%i') AS 小时
FROM crashdata c
LEFT JOIN weatherdata w
ON date_trunc("hour", c.CRASH_DATE)=date_trunc("hour", w.DATE)
WHERE w.HourlyVisibility BETWEEN 0.0 AND 1.0
GROUP BY 小时
ORDER BY 车祸数 DESC
LIMIT 100;
```

低能见度期间，单小时内最高的车祸数为129。有多个事项需要考虑：

- 2014年2月3日是星期一
- 早上8点是高峰时段
- 当时正在下雨（0.12英寸或当小时降水）
- 温度为32华氏度（水的冰点）
- 能见度很差，只有0.25英里，而纽约市的正常值是10英里

#### 在结冰的条件下驾驶怎么办？

水蒸气在40华氏度时可以升华成冰;此查询查看0到40华氏度之间的温度。

```sql
SELECT COUNT(DISTINCT c.COLLISION_ID) AS 车祸数,
       truncate(avg(w.HourlyDryBulbTemperature), 1) AS 温度_F,
       truncate(avg(w.HourlyVisibility), 2) AS 能见度,
       max(w.HourlyPrecipitation) AS 降水量,
       date_format((date_trunc("hour", c.CRASH_DATE)), '%d %b %Y %H:%i') AS 小时
FROM crashdata c
LEFT JOIN weatherdata w
ON date_trunc("hour", c.CRASH_DATE)=date_trunc("hour", w.DATE)
WHERE w.HourlyDryBulbTemperature BETWEEN 0.0 AND 40.5 
GROUP BY 小时
ORDER BY 车祸数 DESC
LIMIT 100;
```

冰冻温度的结果让我有点惊讶，我没想到在一个寒冷的一月天的周日早上城市会有这么多的交通。快速浏览 [weather.com](https://weather.com/storms/winter/news/northeast-storm-rain-snow-wind) 显示，那天发生了一场大风暴，有许多车祸，就像数据中所显示的那样。

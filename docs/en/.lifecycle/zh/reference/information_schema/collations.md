# 排序规则

`collations` 包含了可用的排序规则。

`collations` 中提供了以下字段：

| **字段**          | **描述**                                              |
| ------------------ | ------------------------------------------------------------ |
| COLLATION_NAME     | 排序规则的名称。                                          |
| CHARACTER_SET_NAME | 与排序规则关联的字符集的名称。 |
| ID                 | 排序规则的ID。                                            |
| IS_DEFAULT         | 排序规则是否为其字符集的默认值。  |
| IS_COMPILED        | 字符集是否编译进服务器中。       |
| SORTLEN            | 这与在字符集中表示的字符串进行排序所需的内存量有关。 |

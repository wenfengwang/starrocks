# table_constraints

`table_constraints` 描述了哪些表具有约束条件。

`table_constraints` 提供了以下字段：

|**字段**|**描述**|
|---|---|
|CONSTRAINT_CATALOG|约束所属的目录名称。此值始终为 `def`。|
|CONSTRAINT_SCHEMA|约束所属数据库的名称。|
|CONSTRAINT_NAME|约束的名称。|
|TABLE_SCHEMA|表所属数据库的名称。|
|TABLE_NAME|表的名称。|
|CONSTRAINT_TYPE|约束的类型。|

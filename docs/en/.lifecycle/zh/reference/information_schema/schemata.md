# 数据库模式

`schemata` 提供关于数据库的信息。

`schemata` 中提供了以下字段：

| **字段**                  | **描述**                                              |
| -------------------------- | ------------------------------------------------------------ |
| CATALOG_NAME               | 架构所属的目录名称。此值始终为 `NULL`。 |
| SCHEMA_NAME                | 架构的名称。                                      |
| DEFAULT_CHARACTER_SET_NAME | 架构的默认字符集。                            |
| DEFAULT_COLLATION_NAME     | 架构的默认排序规则。                                |
| SQL_PATH                   | 此值始终为 `NULL`。                                 |

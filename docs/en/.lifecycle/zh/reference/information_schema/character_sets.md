# 字符集

`character_sets` 用于标识可用的字符集。

`character_sets` 包含以下字段：

| **字段**            | **描述**                                              |
| -------------------- | ------------------------------------------------------------ |
| CHARACTER_SET_NAME   | 字符集的名称。                                      |
| DEFAULT_COLLATE_NAME | 字符集的默认排序规则。                 |
| DESCRIPTION          | 字符集的描述。                          |
| MAXLEN               | 存储一个字符所需的最大字节数。 |

# be_bvars

`be_bvars` 提供有关 bRPC 的统计信息。您可以查看 StarRocks 的一些组件的 RPC 延迟、QPS 和其他统计信息。

`be_bvars` 提供以下字段：

| **字段** | **描述**                                              |
| --------- | ------------------------------------------------------------ |
| BE_ID     | bvar 所在的 BE 的ID。                      |
| NAME      | bvar 的名称。                                            |
| DESC      | bvar 的描述，包括重要的统计信息。 |

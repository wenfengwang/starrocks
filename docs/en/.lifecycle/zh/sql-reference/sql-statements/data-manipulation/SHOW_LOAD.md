## 用法注意事项

- SHOW LOAD 语句返回的信息从加载作业的 LoadFinishTime 开始有效期为3天。3天后，无法显示该信息。您可以使用 `label_keep_max_second` 参数修改默认的有效期。

  ```SQL
  ADMIN SET FRONTEND CONFIG ("label_keep_max_second" = "value");
  ```

- 如果 LoadStartTime 字段的值长时间为 `N/A`，表示加载作业堆积严重。建议您减少创建加载作业的频率。
- 加载作业消耗的总时间 = LoadFinishTime - CreateTime。
- 加载作业在 LOAD 阶段消耗的总时间 = LoadFinishTime - LoadStartTime。

## 使用说明

- `SHOW LOAD` 语句返回的信息自加载作业的 `LoadFinishTime` 起 3 天内有效。3 天后，信息将无法显示。您可以使用 `label_keep_max_second` 参数修改默认有效期。

  ```SQL
  ADMIN SET FRONTEND CONFIG ("label_keep_max_second" = "value");
  ```

- 如果 `LoadStartTime` 字段的值长时间为 `N/A`，则说明加载作业严重堆积。我们建议您减少创建加载作业的频率。
- 加载作业消耗的总时间 = `LoadFinishTime` - `CreateTime`。
- 加载作业在 `LOAD` 阶段消耗的总时间 = `LoadFinishTime` - `LoadStartTime`。

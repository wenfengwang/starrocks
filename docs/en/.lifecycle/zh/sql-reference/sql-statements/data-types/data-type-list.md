# 数据类型概览

StarRocks 支持以下数据类型：[数值](#numeric)、[字符串](#string)、[日期](#date)、[半结构化](#semi-structured)和[其他](#others)。您可以在创建表时定义这些类型的列，将数据加载到这些列，以及查询这些数据。

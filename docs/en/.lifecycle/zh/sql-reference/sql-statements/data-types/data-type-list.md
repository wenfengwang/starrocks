# 数据类型概述

StarRocks支持以下数据类型：[数值型](#numeric)、[字符串型](#string)、[日期型](#date)、[半结构化](#semi-structured)型和[其他](#others)数据类型。您可以在创建表、将数据加载到这些列以及查询数据时定义这些类型的列。

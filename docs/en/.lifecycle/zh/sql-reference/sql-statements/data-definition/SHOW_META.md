# 显示元数据

## 描述

查看CBO统计信息的元数据，包括基本统计信息和直方图。

此语句从v2.4版本开始支持。

### 查看基本统计信息的元数据

#### 语法

```SQL
SHOW STATS META [WHERE]
```

此语句返回以下列。

| **列名** | **描述**                                              |
| ---------- | ------------------------------------------------------------ |
| 数据库   | 数据库名称。                                           |
| 表      | 表名。                                              |
| 列    | 列名。                                            |
| 类型       | 统计类型。`FULL`表示完整收集，`SAMPLE`表示抽样收集。 |
| 更新时间 | 当前表的最新统计信息更新时间。     |
| 属性 | 自定义参数。                                           |
| 健康状况    | 统计信息的健康状况。                       |

### 查看直方图的元数据

#### 语法

```SQL
SHOW HISTOGRAM META [WHERE];
```

此语句返回以下列。

| **列名** | **描述**                                              |
| ---------- | ------------------------------------------------------------ |
| 数据库   | 数据库名称。                                           |
| 表      | 表名。                                              |
| 列     | 列名。                                                 |
| 类型       | 统计类型。值为 `HISTOGRAM` 表示直方图。 |
| 更新时间 | 当前表的最新统计信息更新时间。     |
| 属性 | 自定义参数。                                           |

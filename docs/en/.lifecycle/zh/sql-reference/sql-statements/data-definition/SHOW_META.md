# 显示元数据

## 描述

查看 CBO 统计信息的元数据，包括基础统计数据和直方图。

此语句自 v2.4 版本起支持。

### 查看基础统计数据的元数据

#### 语法

```SQL
SHOW STATS META [WHERE]
```

此语句返回以下列。

|**列**|**描述**|
|---|---|
|数据库|数据库名称。|
|表|表名。|
|列|列名。|
|类型|统计数据的类型。`FULL` 表示完整收集，`SAMPLE` 表示抽样收集。|
|UpdateTime|当前表的最新统计更新时间。|
|属性|自定义参数。|
|健康度|统计信息的健康状况。|

### 查看直方图元数据

#### 语法

```SQL
SHOW HISTOGRAM META [WHERE];
```

此语句返回以下列。

|**列**|**描述**|
|---|---|
|数据库|数据库名称。|
|表|表名。|
|列|列名。|
|类型|统计的类型。值为 `HISTOGRAM` 表示直方图。|
|UpdateTime|当前表的最新统计更新时间。|
|属性|自定义参数。|

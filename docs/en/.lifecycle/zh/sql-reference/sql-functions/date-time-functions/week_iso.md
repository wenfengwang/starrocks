# week_iso

## 描述

返回指定日期所在年份的 ISO 标准周数，为一个介于 `1` 至 `53` 的整数。

## 语法

```Haskell
INT WEEK_ISO(DATETIME date)
```

## 参数

`date`：您想要转换的日期。它必须是 DATE 或 DATETIME 类型。

## 示例

以下示例返回日期 `2008-02-20 00:00:00` 所在年份的 ISO 标准周数：

```SQL
MySQL > select week_iso('2008-02-20 00:00:00');
+-----------------------------------+
| week_iso('2008-02-20 00:00:00')   |
+-----------------------------------+
|                                 8 |
+-----------------------------------+
```

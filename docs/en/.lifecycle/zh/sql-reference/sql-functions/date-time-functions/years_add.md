# years_add

## 描述

### 语法

```Haskell
DATETIME YEARS_ADD(DATETIME expr1,INT expr2)
```

向日期添加指定的时间间隔。单位为年。

expr1 参数是有效的日期时间表达式。

expr2 参数是要添加的年数。

## 例

```Plain Text
select years_add('2010-11-30 23:50:50', 2);
+-------------------------------------+
| years_add('2010-11-30 23:50:50', 2) |
+-------------------------------------+
| 2012-11-30 23:50:50                 |
+-------------------------------------+

select years_add('2010-11-30', 2);
+----------------------------+
| years_add('2010-11-30', 2) |
+----------------------------+
| 2012-11-30 00:00:00        |
+----------------------------+
```

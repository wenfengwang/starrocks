# date_format

## 描述

根据指定的格式，将日期转换为字符串。目前支持的字符串最大长度为 128 个字节。如果返回值的长度超过 128，则返回 NULL。

## 语法

```Haskell
VARCHAR DATE_FORMAT(DATETIME date, VARCHAR format)
```

## 参数

- `date` 参数必须是有效的日期或日期表达式。

- `format` 指定日期或时间的输出格式。

以下是可用的格式：

```Plain Text
%a | 缩写的星期几名称 (Sun 到 Sat)
%b | 缩写的月份名称 (Jan 到 Dec)
%c | 数字表示的月份 (0-12)
%D | 以英文后缀的数字表示的日期
%d | 日期的数字表示 (00-31)
%e | 日期的数字表示 (0-31)
%f | 微秒
%H | 小时 (00-23)
%h | 小时 (01-12)
%I | 小时 (01-12)
%i | 分钟 (00-59)
%j | 一年中的日期 (001-366)
%k | 小时 (0-23)
%l | 小时 (1-12)
%M | 完整的月份名称
%m | 月份的数字表示 (00-12)
%p | 上午或下午
%r | 12 小时制时间 (hh:mm:ss 上午或下午)
%S | 秒 (00-59)
%s | 秒 (00-59)
%T | 24 小时制时间 (hh:mm:ss)
%U | 周数 (00-53)，其中周日为一周的第一天
%u | 周数 (00-53)，其中周一为一周的第一天
%V | 周数 (01-53)，其中周日为一周的第一天。与 %X 一起使用。
%v | 周数 (01-53)，其中周一为一周的第一天。与 %x 一起使用。
%W | 完整的星期几名称
%w | 一周中的日期，星期日为 0，星期六为 6
%X | 一年中的周，其中周日为一周的第一天。4 位数字值。与 %V 一起使用。
%x | 一年中的周，其中周一为一周的第一天。4 位数字值。与 %v 一起使用。
%Y | 年份。4 位数字值。
%y | 年份。2 位数字值。
%% | 表示 %。 
```

## 例子

```Plain Text
MySQL > select date_format('2009-10-04 22:23:00', '%W %M %Y');
+------------------------------------------------+
| date_format('2009-10-04 22:23:00', '%W %M %Y') |
+------------------------------------------------+
| Sunday October 2009                            |
+------------------------------------------------+

MySQL > select date_format('2007-10-04 22:23:00', '%H:%i:%s');
+------------------------------------------------+
| date_format('2007-10-04 22:23:00', '%H:%i:%s') |
+------------------------------------------------+
| 22:23:00                                       |
+------------------------------------------------+

MySQL > select date_format('1900-10-04 22:23:00', '%D %y %a %d %m %b %j');
+------------------------------------------------------------+
| date_format('1900-10-04 22:23:00', '%D %y %a %d %m %b %j') |
+------------------------------------------------------------+
| 4th 00 Thu 04 10 Oct 277                                   |
+------------------------------------------------------------+

MySQL > select date_format('1997-10-04 22:23:00', '%H %k %I %r %T %S %w');
+------------------------------------------------------------+
| date_format('1997-10-04 22:23:00', '%H %k %I %r %T %S %w') |
+------------------------------------------------------------+
| 22 22 10 10:23:00 PM 22:23:00 00 6                         |
+------------------------------------------------------------+

MySQL > select date_format('1999-01-01 00:00:00', '%X %V');
+---------------------------------------------+
| date_format('1999-01-01 00:00:00', '%X %V') |
+---------------------------------------------+
| 1998 52                                     |
+---------------------------------------------+

MySQL > select date_format('2006-06-01', '%d');
+------------------------------------------+
| date_format('2006-06-01 00:00:00', '%d') |
+------------------------------------------+
| 01                                       |
+------------------------------------------+

MySQL > select date_format('2006-06-01', '%%%d');
+--------------------------------------------+
| date_format('2006-06-01 00:00:00', '%%%d') |
+--------------------------------------------+
| %01                                        |
+--------------------------------------------+
```

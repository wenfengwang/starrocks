# 年份

## 描述

返回日期中的年份部分，并返回一个从 1000 到 9999 的值。

`date` 参数必须是 DATE 或 DATETIME 类型。

## 语法

```Haskell
INT YEAR(DATETIME date)
```

## 例子

```Plain Text
MySQL > select year('1987-01-01');
+-----------------------------+
| year('1987-01-01 00:00:00') |
+-----------------------------+
|                        1987 |
+-----------------------------+
```

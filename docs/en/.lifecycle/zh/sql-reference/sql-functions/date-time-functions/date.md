# 日期

## 描述

提取日期或日期时间表达式的日期部分。

## 语法

```Haskell
DATE date(DATETIME|DATE expr)
```

## 参数

`expr`：日期或日期时间表达式。

## 返回值

返回 DATE 类型的值。如果输入为 NULL 或无效，则返回 NULL。

## 例子

示例 1：提取日期时间值的日期部分。

```plaintext
SELECT DATE("2017-12-31 11:20:59");
+-----------------------------+
| date('2017-12-31 11:20:59') |
+-----------------------------+
| 2017-12-31                  |
+-----------------------------+
1 行受影响 (0.05 秒)
```

示例 2：提取日期值的日期部分。

```plaintext
SELECT DATE('2017-12-31');
+--------------------+
| date('2017-12-31') |
+--------------------+
| 2017-12-31         |
+--------------------+
1 行受影响 (0.08 秒)
```

示例 3：提取当前时间戳的日期部分。

```plaintext
SELECT DATE(current_timestamp());
+---------------------------+
| date(current_timestamp()) |
+---------------------------+
| 2022-11-08                |
+---------------------------+
1 行受影响 (0.05 秒)

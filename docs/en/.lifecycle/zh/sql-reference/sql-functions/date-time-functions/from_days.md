# from_days

## 描述

返回一个从公元0000年01月01日起的日期。

## 语法

```Haskell
DATE FROM_DAYS(INT N)
```

## 示例

```Plain
MySQL > select from_days(730669);
+-------------------+
| from_days(730669) |
+-------------------+
| 2000-07-03        |
+-------------------+
```

## 关键字

FROM_DAYS,FROM,DAYS

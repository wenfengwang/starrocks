# COT

## 描述

返回以弧度 `x` 表示的角的余切值。如果弧度 `x` 为 `0` 或 `NULL`，则返回 `NULL`。

## **语法**

```SQL
COT(x);
```

## **参数**

`x`：角度的弧度值。支持 DOUBLE 类型。

## **返回值**

返回 DOUBLE 数据类型的值。

## **示例**

```Plaintext
mysql> select cot(3.1415926/2);
+------------------------+
| cot(3.1415926 / 2)     |
+------------------------+
| 2.6794896585028646e-08 |
+------------------------+
1 row in set (0.25 sec)

mysql> select cot(0);
+--------+
| cot(0) |
+--------+
|   NULL |
+--------+
1 row in set (0.03 sec)
```

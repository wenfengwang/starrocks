# 余切

## 描述

返回以弧度为单位的角度 `x` 的余切值。如果以弧度为单位的角度 `x` 为 `0` 或 `NULL`，则返回 `NULL`。

## **语法**

```SQL
COT(x);
```

## **参数**

`x`：以弧度表示的角度。支持 DOUBLE 类型。

## **返回值**

返回 DOUBLE 数据类型的值。

## **例子**

```Plaintext
mysql> select cot(3.1415926/2);
+------------------------+
| cot(3.1415926 / 2)     |
+------------------------+
| 2.6794896585028646e-08 |
+------------------------+
1 行受影响 (0.25 秒)

mysql> select cot(0);
+--------+
| cot(0) |
+--------+
|   NULL |
+--------+
1 行受影响 (0.03 秒)

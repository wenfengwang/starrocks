# ln、dlog1、log

## 描述

计算一个数的自然（以 `e` 为底）对数。

## 语法

```SQL
ln(arg)
log(arg)
dlog1(arg)
```

## 参数

- `arg`：要计算其对数的值。仅支持 DOUBLE 数据类型。

> **注意**
>
> 如果指定的 `arg` 为负数或 0，则 StarRocks 返回 NULL。

## 返回值

返回 DOUBLE 数据类型的值。

## 例子

示例 1：计算 3 的自然对数。

```SQL
mysql> select ln(3);
+--------------------+
| ln(3)              |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 行受影响 (0.00 秒)

mysql> select dlog1(3);
+--------------------+
| dlog1(3)           |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 行受影响 (0.00 秒)

mysql> select log(3);
+--------------------+
| log(3)             |
+--------------------+
| 1.0986122886681098 |
+--------------------+
1 行受影响 (0.00 秒)

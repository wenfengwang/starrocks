# e

## 描述

返回常数 e（自然对数的底数）。

## 语法

```SQL
E();
```

## 参数

此函数不接受任何参数。

## 返回值

返回常数 e，数据类型为 DOUBLE。

## 示例

```Plaintext
mysql> select e();
+-------------------+
| e()               |
+-------------------+
| 2.718281828459045 |
+-------------------+
1 行在集合中 (0.01 秒)
```

# acos

## 描述

计算参数的反余弦值。

## 语法

```Haskell
DOUBLE acos(DOUBLE arg)
```

### 参数

`arg`：只能指定数值。在计算反余弦之前，函数会将数值转换为 DOUBLE 类型。

## 返回值

返回 DOUBLE 数据类型的值。

## 使用说明

如果指定的是非数值，则该函数返回 `NULL`。

## 示例

```Plain
mysql> select acos(-1);
+-------------------+
| acos(-1)          |
+-------------------+
| 3.141592653589793 |
+-------------------+

mysql> select acos(0);
+--------------------+
| acos(0)            |
+--------------------+
| 1.5707963267948966 |
+--------------------+

mysql> select acos(1);
+---------+
| acos(1) |
+---------+
|       0 |
+---------+

mysql> select acos("");
+----------+
| acos('') |
+----------+
|     NULL |
+----------+
```

# π

## 描述

返回圆周率 Pi 的值。

## 语法

```Plain
pi()
```

## 参数

不适用

## 返回值

返回圆周率 Pi 的值，数据类型为 DOUBLE。

## 示例

```Plain
mysql> select pi();
+-------------------+
| pi()              |
+-------------------+
| 3.141592653589793 |
+-------------------+
1 行在集合中 (0.00 秒)
```

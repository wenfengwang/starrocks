# rand, random

## 描述

返回一个介于 0（包含）和 1（不包含）之间的随机浮点数。

## 语法

```Haskell
RAND(x);
```

## 参数

`x`：可选。数据类型为 BIGINT。如果指定了 `x`，此函数将返回一个可重复的随机数。如果未指定 `x`，此函数将返回一个完全随机的数。

## 返回值

返回一个 DOUBLE 类型的值。

## 例子

```Plain Text
select rand();
+--------------------+
| rand()             |
+--------------------+
| 0.9393535880089522 |
+--------------------+
1 行受影响 (0.01 秒)
select rand(3);
+--------------------+
| rand(3)            |
+--------------------+
| 0.6659865964511347 |
+--------------------+
1 行受影响 (0.00 秒)
```

## 关键字

RAND, RANDOM

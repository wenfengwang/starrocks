# rand, random

## 描述

返回一个介于 0（含）和 1（不含）之间的随机浮点数。

## 语法

```Haskell
RAND(x);
```

## 参数

`x`：可选。数据类型为 BIGINT。如果指定了 `x`，则该函数返回一个可重复的随机数。如果未指定 `x`，则该函数返回一个完全随机的数字。

## 返回值

返回 DOUBLE 类型的值。

## 示例

```Plain
select rand();
+--------------------+
| rand()             |
+--------------------+
| 0.9393535880089522 |
+--------------------+
1 row in set (0.01 sec)
select rand(3);
+--------------------+
| rand(3)            |
+--------------------+
| 0.6659865964511347 |
+--------------------+
1 row in set (0.00 sec)
```

## 关键字

RAND, RANDOM

# 双曲正弦

## 描述

计算参数的双曲正弦值。

此函数从 v3.0 版本开始提供支持。

## 语法

```Haskell
DOUBLE sinh(DOUBLE arg)
```

### 参数

`arg`：只能指定一个数值。此函数在计算该值的双曲正弦值之前将数值转换为 DOUBLE 值。

## 返回值

返回 DOUBLE 数据类型的值。

## 使用说明

如果指定非数值，则此函数返回 `NULL`。

## 例子

```Plain
mysql> select sinh(-1);
+---------------------+
| sinh(-1)            |
+---------------------+
| -1.1752011936438014 |
+---------------------+

mysql> select sinh(0);
+---------+
| sinh(0) |
+---------+
|       0 |
+---------+

mysql> select sinh(1);
+--------------------+
| sinh(1)            |
+--------------------+
| 1.1752011936438014 |
+--------------------+

mysql> select sinh("");
+----------+
| sinh('') |
+----------+
|     NULL |
+----------+
```

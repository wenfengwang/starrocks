# sin

## 描述

返回 `x` 的正弦值，其中 `x` 以弧度为单位。

## 语法

```Haskell
SIN(x);
```

## 参数

`x`：支持 DOUBLE 数据类型。

## 返回值

返回 DOUBLE 数据类型的值。

## 示例

```Plain
mysql> select sin(3.14);
+-----------------------+
| sin(3.14)             |
+-----------------------+
| 0.0015926529164868282 |
+-----------------------+
1 row in set (0.21 sec)
```

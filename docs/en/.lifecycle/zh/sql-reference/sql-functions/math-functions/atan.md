# atan

## 描述

计算参数的反正切值。

## 语法

```Haskell
DOUBLE atan(DOUBLE arg)
```

### 参数

`arg`：您只能指定一个数值。此函数在计算该值的反正切值之前将数值转换为 DOUBLE 值。

## 返回值

返回 DOUBLE 数据类型的值。

## 使用说明

如果指定了非数值，则此函数返回 `NULL`。

## 例子

```Plain
mysql> select atan(1);
+--------------------+
| atan(1)            |
+--------------------+
| 0.7853981633974483 |
+--------------------+

mysql> select atan(0);
+---------+
| atan(0) |
+---------+
|       0 |
+---------+

mysql> select atan(-1);
+---------------------+
| atan(-1)            |
+---------------------+
| -0.7853981633974483 |
+---------------------+

mysql> select atan("");
+----------+
| atan('') |
+----------+
|     NULL |
+----------+
```

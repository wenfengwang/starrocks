# 角度

## 描述

将弧度为 `x` 的角度转换为度数。

## 语法

```SQL
DEGREES(x);
```

## 参数

`x`：以弧度为单位的角度。支持 DOUBLE 类型。

## 返回值

返回 DOUBLE 数据类型的值。

## 例子

```Plaintext
mysql> select degrees(3.1415926535898);
+--------------------------+
| degrees(3.1415926535898) |
+--------------------------+
|        180.0000000000004 |
+--------------------------+
1 行受影响 (0.07 秒)

# log10、dlog10

## 描述

计算一个数字的以 10 为底的对数。

## 语法

```SQL
log10(arg)
```

## 参数

- `arg`：要计算其对数的值。仅支持 DOUBLE 数据类型。

> **注意**
>
> 如果 `arg` 被指定为负数或 0，则返回 NULL。

## 返回值

返回一个 DOUBLE 数据类型的值。

## 例

示例 1：计算 100 的以 10 为底的对数。

```Plain
select log10(100);
+------------+
| log10(100) |
+------------+
|          2 |
+------------+
1 行结果（0.02 秒）

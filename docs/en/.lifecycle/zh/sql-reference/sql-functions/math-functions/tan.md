# 正切

## 描述

返回`x`的正切值，其中`x`以弧度为单位。

## 语法

```Haskell
TAN(x);
```

## 参数

`x`：支持DOUBLE数据类型。

## 返回值

返回一个DOUBLE数据类型的值。

## 例子

```Plain
mysql> select tan(3.14);
+-----------------------+
| tan(3.14)             |
+-----------------------+
| -0.001592654936407223 |
+-----------------------+
1 行受影响 (0.12 秒)

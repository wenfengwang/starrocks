# tan

## 描述

返回 `x` 的正切值，其中 `x` 为弧度单位。

## 语法

```Haskell
TAN(x);
```

## 参数

`x`：支持 DOUBLE 数据类型。

## 返回值

返回 DOUBLE 数据类型的值。

## 示例

```Plain
mysql> select tan(3.14);
+-----------------------+
| tan(3.14)             |
+-----------------------+
| -0.001592654936407223 |
+-----------------------+
1 row in set (0.12 sec)
```

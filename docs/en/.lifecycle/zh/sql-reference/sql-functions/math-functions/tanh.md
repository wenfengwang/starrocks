# tanh

## 描述

计算参数的双曲正切。

该函数从v3.0版本开始支持。

## 语法

```Haskell
DOUBLE tanh(DOUBLE arg)
```

### 参数

`arg`：只能指定数值。在计算双曲正切之前，该函数会将数值转换为DOUBLE类型。

## 返回值

返回DOUBLE数据类型的值。

## 使用说明

如果指定的是非数值，则该函数返回`NULL`。

## 示例

```Plain
mysql> select tanh(-1);
+---------------------+
| tanh(-1)            |
+---------------------+
| -0.7615941559557649 |
+---------------------+

mysql> select tanh(0);
+---------+
| tanh(0) |
+---------+
|       0 |
+---------+

mysql> select tanh(1);
+--------------------+
| tanh(1)            |
+--------------------+
| 0.7615941559557649 |
+--------------------+

mysql> select tanh("");
+----------+
| tanh('') |
+----------+
|     NULL |
+----------+
```

# 双曲余弦(cosh)

## 描述

计算参数的双曲余弦值。

此函数从v3.0版本开始得到支持。

## 语法

```Haskell
DOUBLE cosh(DOUBLE arg)
```

### 参数

`arg`：只能指定一个数值。此函数在计算该值的双曲余弦值之前，将数值转换为DOUBLE类型。

## 返回值

返回一个DOUBLE数据类型的值。

## 使用说明

如果指定了非数值的参数，该函数将返回`NULL`。

## 例子

```Plain
mysql> select cosh(-1);
+--------------------+
| cosh(-1)           |
+--------------------+
| 1.5430806348152437 |
+--------------------+

mysql> select cosh(0);
+---------+
| cosh(0) |
+---------+
|       1 |
+---------+

mysql> select cosh(1);
+--------------------+
| cosh(1)            |
+--------------------+
| 1.5430806348152437 |
+--------------------+

mysql> select cosh("");
+----------+
| cosh('') |
+----------+
|     NULL |
+----------+
```

# 余弦

## 描述

计算参数的余弦值。

## 语法

```Haskell
DOUBLE cos(DOUBLE arg)
```

### 参数

`arg`：只能指定一个数值。此函数在计算值的余弦值之前将数值转换为 DOUBLE 值。

## 返回值

返回 DOUBLE 数据类型的值。

## 使用说明

如果指定非数值，则此函数返回 `NULL`。

## 例子

```Plain
mysql> select cos(-1);
+--------------------+
| cos(-1)            |
+--------------------+
| 0.5403023058681398 |
+--------------------+

mysql> select cos(0);
+--------+
| cos(0) |
+--------+
|      1 |
+--------+

mysql> select cos(1);
+--------------------+
| cos(1)             |
+--------------------+
| 0.5403023058681398 |
+--------------------+

mysql> select cos("");
+---------+
| cos('') |
+---------+
|    NULL |
+---------+
```

# square

## 描述

计算一个值的平方。

## 语法

```Haskell
square(arg)
```

### 参数

`arg`：你想要计算平方的值。你只能指定一个数值类型的参数。该函数在计算平方之前会将数值转换为 DOUBLE 类型。

## 返回值

返回 DOUBLE 数据类型的值。

## 使用说明

如果指定的是非数值类型，则该函数返回 `NULL`。

## 示例

示例 1：计算一个数值的平方。

```Plain
mysql>  select square(11);
+------------+
| square(11) |
+------------+
|        121 |
+------------+
```

示例 2：计算一个非数值的平方。返回值为 `NULL`。

```Plain
mysql>  select square('2021-01-01');
+----------------------+
| square('2021-01-01') |
+----------------------+
|                 NULL |
+----------------------+
```
